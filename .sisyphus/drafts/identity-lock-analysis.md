# Draft: 씨네보드 스타일의 쇼츠랩 Identity Lock 개선 분석

## 1. 개요
- **대상**: 쇼츠랩(Shorts Lab) 내 인물 설정 기능
- **현재 이슈**: 데이터 항목은 [이름, 나이, 캐릭터, 의상, 악세서리]로 업데이트되었으나, 인터페이스(UI)가 씨네보드에 비해 불편함
- **목표**: 씨네보드의 우수한 인터페이스 방식을 쇼츠랩의 신규 항목들에 이식하여 사용자 편의성 극대화

## 2. 요구사항 (확정)
- **항목 구성**: 이름, 나이, 캐릭터, 의상, 악세서리
- **인터페이스 스타일**: **씨네보드(Cineboard) 스타일 준수**
  - 카드 형태의 배치
  - 직관적인 아이콘 및 시각적 요소
  - (분석 중인 씨네보드만의 강점 적용)

## 3. 기술적 현황 및 대조 (분석 중)
- **쇼츠랩 UI 위치**: [파일 탐색 중]
- **씨네보드 UI 특징**: `CineboardPanel.tsx` 내 카드 기반 레이아웃, 가상 캐스팅 연동 등
- **차이점**: 쇼츠랩은 단순 입력창 나열일 가능성 높음 -> 씨네보드는 시각적 카드/아코디언 구조

## 4. 개선 아이디어
- 쇼츠랩의 [나이, 의상, 악세서리] 항목을 씨네보드의 카드 UI 내에 '태그' 또는 '아이콘 버튼' 형태로 배치
- 씨네보드의 'Identity Lock(자물쇠)' 개념을 쇼츠랩 항목 전체에 적용
- 인물별 '비주얼 프리뷰(캐스팅 이미지)'를 중심에 두는 레이아웃 유지
