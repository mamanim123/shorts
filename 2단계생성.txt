1. 첫번째 대본 생성을 위한 llm 에게 보낸 프롬프트

[SYSTEM: STRICT JSON OUTPUT ONLY - NO EXTRA TEXT]

당신은 유튜브 쇼츠 대본 전문 작가입니다.
아래 주제/장르/연령에 맞는 대본만 생성하세요.
이미지 프롬프트나 scenes는 절대 생성하지 않습니다.

주제: 골프장에 눈이오면 생기는일
장르: 코미디 / 유머
타겟 연령: 40s

출력은 반드시 아래 JSON 형식 하나만:
{
  "title": "제목",
  "titleOptions": ["옵션1", "옵션2", "옵션3"],
  "scriptBody": "문장1\n문장2... (8~12개 문장)",
  "punchline": "펀치라인",
  "hook": "HOOK 문장",
  "twist": "TWIST 문장",
  "foreshadowing": "복선 문장",
  "narrator": { "slot": "WomanA", "name": "지영" },
  "emotionFlow": "😊 평화 → 😳 당황 → 😱 충격 → 🤦 민망 → 😂 자폭 웃음"
}

규칙:
1) scriptBody는 8~12문장 (줄바꿈으로 구분)
2) 대사는 자연스러운 구어체 한국어
3) scenes/longPrompt/shortPrompt 절대 포함 금지
4) 마크다운 금지, JSON만 출력

2. 대본과 이미지프롬프트 만들때 llm 에게 보낸 프롬프트

[SYSTEM: STRICT JSON OUTPUT ONLY - NO EXTRA TEXT]

당신은 장면 분해 전문가입니다. 아래 "대본 라인"을 그대로 유지하면서 씬 정보를 구조화하세요.
대본을 새로 쓰거나 요약해서 scriptLine을 바꾸지 마세요.

## 필수 규칙
1) JSON만 출력 (설명/마크다운 금지)
2) scenes 개수 = 대본 라인 수 (1:1 매칭)
3) scriptLine은 대본 라인을 **그대로 복사**
4) characterIds는 아래 목록의 ID만 사용 (없으면 빈 배열 [])
5) summary/action/background는 짧고 명확한 영어 묘사로 작성

## 대본 라인
1. 어제 골프장 예약해놨는데 아침에 일어나니까 눈이 펑펑 오는 거야.
2. 근데 골프장에서 문자가 온 거지. '오픈합니다' 이러면서.
3. 아니 눈이 이렇게 오는데 골프를 치라고? 했는데 취소하면 위약금이라는 거야.
4. 그래서 뭐 어쩔 수 없이 갔지. 도착했는데 코스가 온통 새하얀 거야.
5. 캐디님이 주황색 골프공을 건네주시더라고. '오늘은 이거 쓰셔야 찾습니다' 하면서.
6. 1번홀 티샷을 날렸는데, 공이 눈 위에서 통통 튀면서 평소보다 30미터를 더 가는 거야.
7. 아 이거 오늘 드라이버 폼 미쳤다 싶었는데, 그냥 눈 위에서 미끄러진 거더라고.
8. 그린에서는 퍼팅할 때 공이 눈 때문에 중간에 멈춰버리고.
9. 캐디님이 '오늘은 그냥 눈사람 만드시는 게 나을 것 같습니다' 하시더라.
10. 결국 9홀만 돌고 클럽하우스에서 소주나 마셨지 뭐야.
11. 근데 신기한 건 나 말고도 이 날씨에 온 사람이 세 팀이나 더 있더라고.
12. 다들 위약금이 아까워서 온 거래. 우리 진짜 돈 아끼려다가 폐렴 걸리겠더라.

## 캐릭터 ID 목록
- WomanA (지영) / Woman A
- WomanB (혜경) / Woman B
- ManA (준호) / Man A
- ManB (민수) / Man B
- WomanD (지우) / Woman D

## 출력 JSON 스키마
{
  "title": "string",
  "scenes": [
    {
      "sceneNumber": 1,
      "scriptLine": "원문 그대로",
      "summary": "short english summary",
      "action": "short english action",
      "background": "short english background",
      "shotType": "원샷/투샷/쓰리샷",
      "characterIds": ["WomanA"]
    }
  ]
}
