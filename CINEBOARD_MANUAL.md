# 🎬 AI 씨네보드 (AI Cineboard) 사용 설명서

씨네보드는 대본을 바탕으로 시네마틱한 장면(Scene)별 프롬프트를 생성하고, 실제 이미지까지 자동으로 뽑아주는 스토리보드 제작 도구입니다.

## 🚀 주요 기능

### 1. 설정 및 대본 입력
- **비주얼 스타일**: 감성사극, K-드라마, 누아르, 동화 등 원하는 영상의 무드를 선택합니다.
- **엔진 선택**: 나노(빠름) 또는 프로(정밀) 엔진을 선택합니다.
- **비율 선택**: 16:9(롱폼) 또는 9:16(쇼츠) 비율을 지정합니다.
- **대본 입력**: 텍스트를 직접 입력하거나 파일을 드래그 앤 드롭하여 넣을 수 있습니다.

### 2. Identity Lock (인물 고정)
- **인물 자동 추출**: 대본을 분석하여 등장인물을 자동으로 찾아냅니다.
- **캐스팅 이미지 업로드**: 각 인물별로 참조할 이미지(뼈대, 색감용)를 업로드할 수 있습니다. 
  - *업로드 시 AI(Gemini 등)가 해당 이미지를 실제로 분석하여 프롬프트에 반영합니다.*
- **특징 고정**: 인물별 세부 특징(예: 왼쪽 눈 밑 점, 갈색 갓 등)을 메모해두면 모든 장면에 일관되게 적용됩니다.
- **누락 인물 찾기**: 대본 분석에서 빠진 인물을 후보 사전을 기반으로 재검색합니다.

### 3. 씨네보드 생성 및 뷰어
- **프롬프트 생성**: 대본을 바탕으로 씬별 요약, 카메라 워킹, 영문/국문 시각 프롬프트를 JSON 형태로 만듭니다.
- **결과 확인**: 생성된 결과는 즉시 '생성 결과' 탭에서 확인할 수 있습니다.
- **그림 생성**: 
  - 각 장면 옆의 **[그림 생성]** 버튼을 눌러 실제 이미지를 생성합니다.
  - 상단의 **[전체 그림 생성]** 버튼을 누르면 모든 장면의 이미지를 순차적으로 자동 생성합니다.
- **자동 저장**: 생성된 모든 텍스트 결과는 `generated_scripts/` 폴더에, 이미지는 `generated_scripts/images/` 폴더에 자동으로 테마별로 정리되어 저장됩니다.

## 🛠 작업 흐름 가이드

1. **대본 준비**: 제작할 영상의 시나리오를 준비합니다.
2. **인물 세팅**: 대본을 넣고 추출된 인물들에게 'Identity Lock' 설정을 합니다. (캐스팅 이미지 업로드 권장)
3. **씨네보드 생성**: [씨네보드 생성] 버튼을 눌러 전체 구성안을 만듭니다.
4. **이미지 제작**: 생성 결과 탭에서 [전체 그림 생성]을 눌러 스토리보드 이미지를 완성합니다.
5. **결과 활용**: 저장된 JSON 파일이나 텍스트 결과를 영상 제작(Sora, Runway, Pika 등)의 프롬프트로 활용하세요.
