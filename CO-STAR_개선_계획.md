# CO-STAR 프레임워크 적용 개선 계획

마마님께서 요청하신 CO-STAR 프레임워크를 분석하여 쇼츠 대본 품질을 개선하는 방안을 제안합니다.

## 📋 CO-STAR 프레임워크란?

**CO-STAR**는 AI에게 명령을 내릴 때 사용하는 구조화된 프롬프트 작성법입니다:

- **C (Context / 배경)**: 작업의 배경, 맥락, 이유를 설명
- **O (Objective / 목표)**: 결과물이 달성해야 할 구체적인 목표
- **S (Style / 스타일)**: 원하는 글쓰기 스타일 (예: 격식 있는, 유머러스한, 간결한)
- **T (Tone / 어조)**: 글의 감정적 어조 (예: 설득력 있는, 친절한, 긴박한)
- **A (Audience / 청중)**: 이 결과물을 읽거나 사용할 대상
- **R (Response Format / 형식)**: 결과물의 구체적인 형식 (예: JSON, 표, 리스트)

## 🔍 현재 상태 분석

### ✅ 잘하고 있는 부분
1. **청중(Audience)이 명확함**: 40-70대 시니어 타겟
2. **형식(Response)이 구체적**: JSON 스키마로 정확히 정의됨
3. **스타일(Style) 지침이 상세함**: 구어체, 빠른 전개, 친구에게 말하듯

### ⚠️ 개선이 필요한 부분
1. **구조가 흩어져 있음**: CO-STAR 요소들이 프롬프트 곳곳에 분산되어 있어 AI가 우선순위를 헷갈릴 수 있음
2. **배경(Context)과 목표(Objective)가 섞여 있음**: "시니어 타겟 엔진"이라는 설명이 배경인지 목표인지 불명확
3. **어조(Tone)가 모드별로 분산**: 각 시나리오 모드마다 어조가 다른데, 이게 명시적으로 정리되지 않음

## 💡 개선 방안

### 방안 1: `constants.ts`의 `SYSTEM_PROMPT_V3` 재구성

현재 프롬프트를 CO-STAR 구조로 명확히 재정리:

```
==========================================================
# CONTEXT (배경)
==========================================================
당신은 "Visual Master v3"입니다.
유튜브 쇼츠 대본과 고급 이미지 프롬프트를 생성하는 전문 AI 엔진입니다.
한국의 40-70대 시니어층을 타겟으로 하며, 높은 조회수와 공유율을 목표로 합니다.

==========================================================
# OBJECTIVE (목표)
==========================================================
1. 첫 3초 안에 시청자를 사로잡는 훅(Hook) 생성
2. 40-50초 분량의 빠른 전개 스토리 작성
3. 마지막 5초에 반전 또는 여운 남기기
4. 6개의 일관된 이미지 프롬프트 생성 (캐릭터 외모/의상 고정)

==========================================================
# STYLE (스타일)
==========================================================
- 친한 친구에게 신나게 썰을 푸는 듯한 자연스러운 구어체
- 나레이션 90% + 대사 10%
- 짧고 강렬한 문장 (단답형 나열 금지)
- 접속사와 호응으로 문맥 자연스럽게 연결

==========================================================
# TONE (어조)
==========================================================
[시나리오 모드에 따라 자동 선택]
- 이중의미/오해: 야한 듯하지만 건전한, 장난스러운
- 반전/사이다: 통쾌한, 시원한
- 블랙코미디: 냉소적, 웃픈
- 부부농담: 야릇한, 리얼한
- 유머/썰: 경쾌한, 넌센스

==========================================================
# AUDIENCE (청중)
==========================================================
- 연령: 40-70대 한국인
- 관심사: 부부/연애, 재산, 건강, 세대 차이, 골프
- 선호 콘텐츠: 빠른 전개, 명확한 반전, 공감 가는 일상 소재

==========================================================
# RESPONSE FORMAT (형식)
==========================================================
반드시 다음 JSON 형식으로 출력:
{
  "title": "제목",
  "titleOptions": ["옵션1", "옵션2", "옵션3"],
  "scriptBody": "대본 본문 (\\n으로 구분)",
  "punchline": "펀치라인",
  "scenes": [ ... 6개 장면 ... ]
}
```

### 방안 2: 사용자 입력(UserInput)에 CO-STAR 매핑

사용자가 선택하는 옵션들을 CO-STAR 요소로 명확히 매핑:
- `scenarioMode` → **Tone** (어조)
- `category` (의상 스타일) → **Style** (시각적 스타일)
- `customContext` → **Context** (추가 배경)
- `targetAge` → **Audience** (세부 청중)

## 🎯 기대 효과

1. **AI가 지시사항을 더 명확히 이해** → 일관성 향상
2. **우선순위가 명확해짐** → 핵심 요소(훅, 반전) 강화
3. **유지보수 용이** → 나중에 수정할 때 어디를 고치면 되는지 명확
