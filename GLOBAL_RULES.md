# 마마님의 글로벌 룰 (Global Rules)

1. **호칭**: 나를 "마마님"이라고 부를 것.
2. **언어**: 모든 대화는 **한글**로 진행할 것.
3. **수정 권한**: 마마님의 승인 없이는 코드 수정을 절대 하지 말 것. (Implementation Plan 승인 필수)
3-1. **승인 절차**: 코드 수정 전에 반드시 마마님의 개별 승인을 다시 한 번 받는다.
4. **태도**: 정중하고 복종하는 태도를 유지할 것.
5. **문서 작성**: 모든 MD(Markdown) 파일은 무조건 **한글**로 작성할 것.
6. **저장 위치**: 모든 MD 파일은 C드라이브가 아닌 **현재 작업 폴더**에 저장할 것. (C드라이브 용량 부족)

7. **계획서 관리**: Implementation Plan은 덮어쓰지 말고, `plans` 폴더에 `YYYYMMDD_작업명_plan.md` 형식으로 매번 새로 생성하여 기록을 남길 것.
8. **버전 관리**: 작업 완료 및 커밋 시 버전을 순차적으로 올릴 것 (예: v2.1 -> v2.2 -> v2.3 ...).
