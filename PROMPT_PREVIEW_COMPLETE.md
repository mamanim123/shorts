**추가할 코드**:
```tsx
{/* 프롬프트 미리보기 버튼 */}
<button
  onClick={() => setShowPromptPreview(true)}
  className="bg-gray-200 dark:bg-gray-700 hover:bg-gray-300 dark:hover:bg-gray-600 text-gray-800 dark:text-gray-200 px-4 py-2 rounded-lg font-semibold transition-colors flex items-center gap-2"
>
  <Sparkles className="w-4 h-4" />
  프롬프트 미리보기
</button>
```

**찾는 방법**:
1. `youtube-shorts-script-generator.tsx` 파일 열기
2. "대본 생성" 또는 "생성하기" 버튼 찾기
3. 그 버튼 옆에 위 코드 추가
