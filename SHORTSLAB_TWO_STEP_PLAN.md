# ShortsLab 2단계 생성 작업 계획

## 목표
- AI 대본 생성과 분리된 2단계 생성(대본 → 이미지프롬프트) 흐름을 완성한다.
- 기존 AI 대본 생성 로직과 충돌 없이, 수동대본/AI대본 모두 품질을 유지한다.

## 작업 현황
### 완료됨
- 대본만 생성하는 프롬프트 생성기 추가 (`buildLabScriptOnlyPrompt`).
- 2단계 생성 핸들러 추가: 대본 생성 → 씬 분해 → 이미지프롬프트 구성.
- 캐릭터별 랜덤 의상/악세서리 매핑 로직 추가 (중복/동일화 방지).
- 겨울 토글 ON 시: 긴팔 변환 + 겨울 악세서리 병합 적용.
- 대본 기반 캐릭터 추출 단계 추가 (라인별 캐릭터 매핑).
- 프롬프트 검증/보정(`validateAndFixPrompt`) 연동.

### 남은 작업
- 수동 점검: 2단계 생성 시 미리보기/씬 프롬프트 품질 확인.
