## 시작하기

### 필수 요구사항

#### 1. 환경 변수 설정
```bash
# .env 파일에 추가
GEMINI_API_KEY=your_api_key_here
```

#### 2. FFmpeg 설치 (영상 합성용)
```bash
# Windows (Chocolatey)
choco install ffmpeg

# 또는 수동 설치
# https://ffmpeg.org/download.html
```

#### 3. 서버 실행
```bash
# 프로젝트 루트에서
npm install
npm run dev
```

#### 4. AI Studio 접속
- 브라우저에서 `http://localhost:3000` 접속
- "AI Studio" 탭 클릭

## 기본 모드

### 1. 이미지 퓨전 & 수정

**용도**: 여러 이미지를 조합하여 새로운 이미지 생성

**사용 방법**:
1. "이미지 퓨전 & 수정" 버튼 클릭
2. 참조 이미지 업로드 (최대 4개)
3. 각 이미지에 이름 지정 (예: "character", "background")
4. 프롬프트 입력 (예: "Combine character with background")
5. "생성하기" 버튼 클릭

**예시**:
```
이미지 1: 캐릭터 사진 (이름: "character")
이미지 2: 배경 사진 (이름: "background")
프롬프트: "Place character in the background scene"
```

**결과**: 캐릭터가 배경에 합성된 이미지

---

### 2. 텍스트로 이미지 생성

**용도**: 텍스트 프롬프트만으로 이미지 생성

**사용 방법**:
1. "텍스트로 이미지 생성" 버튼 클릭
2. 프롬프트 입력 (최대 4개)
3. "생성하기" 버튼 클릭

**프롬프트 작성 팁**:
- 구체적으로 작성 (색상, 스타일, 분위기 등)
- 예: "A beautiful sunset over the ocean, warm colors, photorealistic"
- 한글도 가능: "바다 위의 아름다운 일몰, 따뜻한 색감, 사실적인 사진"

**예시**:
```
프롬프트 1: "A cute cat sitting on a windowsill, soft lighting"
프롬프트 2: "Modern minimalist living room, white walls, natural light"
```

---

### 3. 동영상 생성

**용도**: 텍스트 프롬프트로 짧은 동영상 생성

**사용 방법**:
1. "동영상 생성" 버튼 클릭
2. (선택) 참조 이미지 업로드
3. 동영상 프롬프트 입력
4. "생성하기" 버튼 클릭

**예시**:
```
프롬프트: "A dragon flying through the night sky"
```

**결과**: 4초 동영상 (MP4)

## Phase 1: 쇼츠 제작 모드

### 4. 📷 카메라 컨트롤

**용도**: 전문적인 카메라 설정으로 이미지 생성

**사용 방법**:
1. "📷 카메라 컨트롤" 버튼 클릭
2. 카메라 설정 선택:
   - **샷 크기**: ECU, CU, MCU, MS, MLS, LS, ELS
   - **앵글**: Eye-level, High angle, Low angle, Dutch angle, OTS
   - **무브먼트**: Static, Pan, Tilt, Dolly, Tracking, Crane
   - **조명**: Natural, Golden hour, Blue hour, Studio, Side, Backlit, Soft
3. 또는 빠른 프리셋 선택 (드라마틱, 자연스러움, 시네마틱, 인물 사진)
4. 프롬프트 입력
5. "생성하기" 버튼 클릭

**빠른 프리셋**:
- **드라마틱**: Low angle + Dramatic lighting
- **자연스러움**: Eye-level + Natural daylight
- **시네마틱**: Medium shot + Cinematic lighting
- **인물 사진**: Close-up + Soft lighting

**예시**:
```
설정:
- 샷 크기: Medium Shot (MS)
- 앵글: Eye-level
- 무브먼트: Static
- 조명: Natural daylight

프롬프트: "A woman standing in a park"
```

**결과**: 전문적인 카메라 설정이 적용된 이미지

### 5. 🎬 프레임 만들기

**용도**: 캐릭터 일관성을 유지하며 연속 장면 생성 (쇼츠용)

**사용 방법**:
1. "🎬 프레임 만들기" 버튼 클릭
2. 참조 캐릭터 이미지 업로드
3. "캐릭터 일관성 유지" 체크
4. 공통 배경 설정 (예: "modern office")
5. 씬 개수 선택 (2-12개)
6. 각 씬별 프롬프트 입력
7. "프레임 생성하기" 버튼 클릭

**예시 (6컷 쇼츠)**:
```
캐릭터: 여성 인물 사진
배경: "modern office"
씬 개수: 6

씬 1: "Woman entering the office, morning"
씬 2: "Woman sitting at desk, working on computer"
씬 3: "Woman having coffee break, smiling"
씬 4: "Woman in meeting with colleagues"
씬 5: "Woman presenting on whiteboard"
씬 6: "Woman leaving office, evening"
```

**결과**: 동일한 캐릭터가 등장하는 6개 프레임

**팁**:
- 참조 이미지는 명확한 인물 사진 사용
- 각 씬 프롬프트는 구체적으로 작성
- 배경은 일관성 있게 유지

---

### 6. 🎥 신 만들기

**용도**: 생성된 프레임을 영상으로 합성 (9:16 쇼츠)

**사용 방법**:
1. "🎥 신 만들기" 버튼 클릭
2. 사용할 프레임 선택 (클릭)
3. 순서 조정 (↑ ↓ 버튼)
4. 영상 설정:
   - **프레임 표시 시간**: 1-10초 (권장: 3초)
   - **트랜지션**: 없음, 페이드, 슬라이드, 줌
   - **트랜지션 시간**: 0.1-2초 (권장: 0.5초)
   - **카메라 효과**: 없음, 줌 인, 줌 아웃, 팬 오른쪽, 팬 왼쪽
5. (선택) 배경 음악 업로드 (MP3)
6. (선택) 음악 볼륨 조절 (0-100%)
7. "🎥 영상 생성하기" 버튼 클릭

**예시 설정**:
```
프레임: 6개 선택
표시 시간: 3초
트랜지션: 페이드 (0.5초)
카메라 효과: 줌 인
배경 음악: background_music.mp3
음악 볼륨: 50%
```

**예상 영상 길이**:
```
6개 프레임 × 3초 = 18초
5개 트랜지션 × 0.5초 = 2.5초
총 길이 = 18 - 2.5 = 15.5초
```

**결과**: 9:16 비율의 MP4 영상 (1080x1920)

**팁**:
- 쇼츠는 15-60초가 적당
- 트랜지션은 부드러운 페이드 추천
- 배경 음악은 저작권 프리 음악 사용

### 7. 🔍 이미지 분석기

**용도**: 이미지에서 상세 프롬프트 추출

**사용 방법**:
1. "🔍 이미지 분석기" 버튼 클릭
2. 분석할 이미지 업로드
3. "🔍 이미지 분석하기" 버튼 클릭
4. 분석 결과 확인:
   - 🎨 스타일
   - 😊 분위기
   - 🌈 색상
   - 📐 구도
   - 💡 조명
   - ✨ 세부사항
5. 필요한 섹션 복사 또는 전체 복사

**활용 방법**:
- 참조 이미지의 스타일 분석
- 프롬프트 작성 시 참고
- 다른 모드에서 재사용

**예시**:
```
입력: 인물 사진

출력:
Style: Photorealistic portrait, professional photography
Mood: Confident, elegant, approachable
Colors: Warm skin tones, soft pastels, neutral background
Composition: Rule of thirds, subject centered
Lighting: Soft studio lighting, three-point setup
Details: Sharp focus on eyes, bokeh background
```

### 8. 🎤 보이스 스튜디오

**용도**: 텍스트를 음성으로 변환 (TTS)

**사용 방법**:
1. "🎤 보이스 스튜디오" 버튼 클릭
2. 텍스트 입력
3. 음성 설정:
   - **목소리**: 여성/남성, Standard/Wavenet (8종)
   - **감정**: 중립, 기쁨, 슬픔, 화남, 흥분
   - **속도**: 0.5x - 2.0x (권장: 1.0x)
   - **음높이**: -20 ~ +20 (권장: 0)
4. "🎤 음성 생성하기" 버튼 클릭
5. 재생 및 다운로드

**감정 프리셋**:
- **중립** (😐): 일반적인 내레이션
- **기쁨** (😊): 밝고 긍정적인 톤
- **슬픔** (😢): 차분하고 감성적인 톤
- **화남** (😠): 강렬하고 빠른 톤
- **흥분** (🤩): 에너지 넘치는 톤

**예시**:
```
텍스트: "안녕하세요! 오늘은 정말 멋진 날씨네요."
목소리: 여성 3 (자연스러움)
감정: 기쁨
속도: 1.0x
음높이: 0
```

**활용**:
- 쇼츠 영상 내레이션
- 오디오북
- 안내 방송

## Phase 2: 이미지 편집 모드

### 9. 🎨 스타일 따라하기

**용도**: 참조 이미지의 스타일을 다른 이미지에 적용

**사용 방법**:
1. "🎨 스타일 따라하기" 버튼 클릭
2. 소스 이미지 업로드 (스타일을 적용할 이미지)
3. 스타일 참조 이미지 업로드 (따라할 스타일)
4. 스타일 강도 조절 (10-100%)
   - 10-40%: 자연스러움
   - 50-70%: 균형 (권장)
   - 80-100%: 강렬함
5. "🎨 스타일 적용하기" 버튼 클릭

**활용 예시**:
- 사진에 유화 스타일 적용
- 만화 스타일 적용
- 특정 색감 적용

**예시**:
```
소스: 일반 인물 사진
스타일: 고흐의 별이 빛나는 밤 (유화)
강도: 70%
```

**결과**: 유화 스타일이 적용된 인물 사진

## 완전 자동화 워크플로우

### 쇼츠 영상 제작 (9.5분)

#### 1단계: 대본 생성 (1분)
- 쇼츠 생성기에서 주제 입력
- 6컷 대본 자동 생성

#### 2단계: 캐릭터 준비 (10초)
- 참조 캐릭터 이미지 1장 업로드

#### 3단계: 카메라 설정 (30초)
- 📷 카메라 컨트롤 모드
- 빠른 프리셋 선택 또는 수동 설정

#### 4단계: 프레임 생성 (3분)
- 🎬 프레임 만들기 모드
- 6개 씬 프롬프트 입력
- 캐릭터 일관성 유지 체크
- 프레임 생성

#### 5단계: 이미지 퀄리티 향상 (2분)
필요에 따라 선택:
- 🎨 스타일 적용

## FAQ

### Q1: 이미지 생성이 실패합니다.
**A**: 다음을 확인하세요:
- GEMINI_API_KEY가 올바르게 설정되었는지
- 프롬프트가 정책 위반이 아닌지
- 인터넷 연결 상태

### Q2: 영상 생성이 안 됩니다.
**A**: FFmpeg가 설치되어 있는지 확인하세요:
```bash
ffmpeg -version
```

### Q3: 캐릭터 일관성이 유지되지 않습니다.
**A**: 
- 참조 이미지를 명확한 인물 사진으로 사용
- "캐릭터 일관성 유지" 체크 확인
- 각 씬 프롬프트에 캐릭터 특징 포함

### Q4: 음성이 생성되지 않습니다.
**A**: 
- 브라우저가 Web Speech API를 지원하는지 확인
- Chrome 또는 Edge 브라우저 사용 권장

### Q5: 프레임 생성이 너무 느립니다.
**A**: 
- 씬 개수를 줄이세요 (6개 권장)
- 프롬프트를 간결하게 작성
- 네트워크 속도 확인

## 문제 해결

### 이미지 생성 오류

**증상**: "Image generation failed" 오류

**해결 방법**:
1. API 키 확인
2. 프롬프트 수정 (정책 위반 가능성)
3. 이미지 크기 확인 (너무 크면 실패)
4. 재시도

### 영상 합성 오류

**증상**: "Scene creation failed" 오류

**해결 방법**:
1. FFmpeg 설치 확인
2. 프레임 개수 확인 (최소 2개)
3. 임시 폴더 권한 확인
4. 디스크 공간 확인

### 음성 생성 오류

**증상**: 음성이 재생되지 않음

**해결 방법**:
1. 브라우저 변경 (Chrome 권장)
2. 텍스트 길이 확인 (너무 길면 끊김)
3. 볼륨 확인

### 성능 문제

**증상**: 느린 처리 속도

**해결 방법**:
1. 프레임 개수 줄이기
2. 이미지 해상도 낮추기
3. 브라우저 캐시 삭제
4. 다른 탭 닫기

## 팁 & 트릭

### 프롬프트 작성 팁

1. **구체적으로 작성**
   - 나쁨: "a person"
   - 좋음: "a young woman with long brown hair, wearing a red dress"

2. **스타일 명시**
   - "photorealistic", "cartoon style", "oil painting"

3. **조명 설명**
   - "soft lighting", "dramatic shadows", "golden hour"

4. **분위기 표현**
   - "peaceful", "energetic", "mysterious"

### 쇼츠 제작 팁

1. **길이**: 15-60초가 적당
2. **트랜지션**: 부드러운 페이드 추천
3. **음악**: 저작권 프리 음악 사용
4. **자막**: 외부 도구에서 추가

### 이미지 편집 팁

1. **보정 강도**: 60-70%가 자연스러움
2. **자연스러움**: 항상 80% 이상 유지
3. **색상 변경**: 질감 유지 옵션 체크
4. **포즈 적용**: 정확도 80% 권장
