# 목표
- 2단계 생성 흐름(대본 → 인물/성별 추출 → 이미지 프롬프트/씬 분해)에 맞춰 각 프롬프트를 분리 관리한다.
- 각 프롬프트를 수정/저장하면 다음 생성부터 즉시 반영된다.
- 백업/복구로 프롬프트 규칙을 안전하게 관리한다.

# 체크리스트
- [ ] 2단계 규칙을 3개 프롬프트로 분리 (대본/인물/이미지)
- [ ] 각 프롬프트 편집 UI 제공 (개별 textarea)
- [ ] 저장 시 즉시 적용되는지 확인
- [ ] 백업 2개 제한 유지
- [ ] 복구 시 즉시 UI 반영 확인
- [ ] 2단계 생성 요청에 수정된 프롬프트가 포함되는지 확인
