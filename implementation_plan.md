# � 검색 필터 확장 계획

마마님, 요청하신 **조회수 필터**와 **날짜 옵션**을 추가하여 검색 기능을 더 강력하게 만들겠습니다.

## 1. �️ 조회수 필터 추가 (신규 기능)
유튜브 검색 API는 "최소 조회수" 설정을 직접 지원하지 않습니다. 따라서 **"일단 검색해온 뒤, 조회수 낮은 건 화면에서 가리는 방식"**으로 구현하겠습니다.

- **옵션 목록**:
  - 전체 (제한 없음)
  - 1만 회 이상
  - 10만 회 이상
  - 30만 회 이상
  - 50만 회 이상
  - 100만 회 이상

## 2. � 날짜 필터 옵션 추가
기존 날짜 필터에 빠져있던 옵션을 추가합니다.
- **추가 옵션**:
  - 3개월 (최근 트렌드 파악용)
  - 6개월 (반기 트렌드 파악용)

## 🛠️ 수정할 파일
- `components/YoutubeSearchPanel.tsx`:
  - `minViewCount` 상태(State) 추가
  - 필터 UI에 `<select>` 추가 (조회수)
  - 날짜 `<select>`에 옵션 추가
  - 결과 필터링 로직(`filteredResults`)에 조회수 조건 추가
