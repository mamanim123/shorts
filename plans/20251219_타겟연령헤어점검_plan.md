# 작업 개요
- 목표: 쇼츠 생성기 타겟 연령 선택 기본값 보장, 헤어스타일 변경 불가 원인 파악
- 포함: 프롬프트 생성 시 타겟 연령 미선택 시 기본값 적용, 헤어스타일 고정 규칙 분석 및 보고
- 제외: UI 대규모 개편, 헤어 커스터마이징 기능 추가 구현

# 작업 단계
1) 타겟 연령 기본값 적용 지점 확인 및 코드 수정 (빈 값/미선택 시 기본값 주입)
2) 헤어스타일 관련 엔진 프롬프트(럭셔리 V3/샤베트 등) 규칙 검토 및 비교 결과 정리

# 리스크 및 대응
- 프롬프트 문자열 수정 시 오타/누락 위험 → 최소 변경으로 기본값 주입
- 헤어 규칙은 엔진별 차이가 있어 가이드로 명확히 설명
