# 샤베트 엔진 프롬프트 하드코딩 수정 계획

## 1. 목표

## 2. 현황 분석
현재 `engine_config.json`에는 3개의 샤베트 엔진 데이터가 존재합니다.

1.  **`CUSTOM_1765968277784`**: "타깃 연령대 초·중반" (하드코딩됨) ❌
2.  **`CUSTOM_1765973385899`**: "타깃 연령대 초·중반" (하드코딩됨) ❌
3.  **`CUSTOM_1766109481051`**: "시스템에서 설정된 나이" (정상) ✅

## 3. 수정 계획
