# 샤베트 엔진 대본 구조 개선안

**제안자**: Mama-nim  
**날짜**: 2025-12-23

## 📋 Mama-nim의 제안 구조

### 1. 강한 훅
- 시청자를 멈추게 만드는 첫 문장

### 2. 상황 깔기 (장소+관계+왜 거기 있는지)
- **핵심**: 설명 말고 '보이는 장면' 위주

### 3. 상상 증폭 구간
- **말 안 해도 느껴지는 디테일 3개**
- 예시: 거리, 시선, 타이밍, 말투, 체형, 몸매, 의상라인

### 4. 착각 최고조
- **시청자가 혼자 결론 내리게 만드는 문장 1개**
- 시청자가 음란마귀가 되게 하기

### 5. 반전
- 완전히 다른 이유/목적/관계 공개
- **설명 없이 한 문장으로 끝**

## 💡 보충 및 개선 제안

### ✅ 강점 분석

1. **"보이는 장면" 위주** → 영상 매체에 최적화 ⭐
2. **디테일 3개 규칙** → 구체적이고 실행 가능 ⭐
3. **시청자가 혼자 결론 내리게** → 참여도 극대화 ⭐
4. **설명 없이 한 문장으로 끝** → 여운 극대화 ⭐

### 🔧 개선 제안

#### 1단계: 강한 훅
**현재**: 강한 훅

**개선안**:
```
1. 강한 훅 (Hook)
   - 형식: 질문형 OR 충격 선언형 OR 금지된 것 암시
   - 예시: 
     * "이거 보고 아무 생각 안 드신 분?"
     * "골프장에서 절대 하면 안 되는 거 했어요"
     * "남편 몰래 골프장 갔다가..."
   - 금지: 평범한 서술 ("오늘 골프장에 갔어요" ❌)
```

#### 2단계: 상황 깔기
**현재**: 상황 깔기 (장소+관계+왜 거기 있는지) - 보이는 장면 위주

**개선안**:
```
2. 상황 깔기 (Scene Setting)
   - 핵심: "말하지 말고 보여줘" (Show, Don't Tell)
   - 필수 요소: 장소 + 관계 + 이유
   - 형식: 시각적 묘사 1~2문장
   - 예시:
     * ❌ "골프장에 남편과 함께 갔어요"
     * ✅ "새벽 6시 골프장, 남편은 벌써 저 멀리 티샷 준비 중이고"
   - 포함 금지: 감정 설명, 의도 설명
```

#### 3단계: 상상 증폭 구간
**현재**: 말 안 해도 느껴지는 디테일 3개

**개선안**:
```
3. 상상 증폭 구간 (Imagination Amplifier)
   - 규칙: 정확히 3개의 디테일 (더도 말고 덜도 말고)
   - 선택 풀: 거리, 시선, 타이밍, 말투, 체형, 몸매, 의상라인, 손동작, 호흡, 향기
   - 배치 순서: 
     1) 물리적 디테일 (거리/시선/의상라인)
     2) 행동 디테일 (타이밍/말투/손동작)
     3) 암시 디테일 (체형/몸매/호흡)
   - 예시:
     * "거리는 팔 하나 정도, 시선은 계속 내 입술 쪽, 목소리는 평소보다 한 톤 낮더라고"
   - 금지: 직접적인 성적 묘사, 결론 유도
```

#### 4단계: 착각 최고조
**현재**: 시청자가 혼자 결론 내리게 만드는 문장 1개

**개선안**:
```
4. 착각 최고조 (Peak Misunderstanding)
   - 목표: 시청자가 "아 이거 그거구나..." 하게 만들기
   - 형식: 애매모호하지만 강력한 암시 1문장
   - 기법:
     * 이중 의미 단어 사용 ("꽉 잡아", "깊이 들어가", "천천히 빼")
     * 상황 묘사만 하고 해석은 시청자에게
     * 숨소리/한숨/신음 같은 소리 묘사
   - 예시:
     * "그때 내가 한숨 쉬면서 이렇게 말했어, 이거 빼기 진짜 힘들다고"
   - 주의: 너무 노골적이면 안 됨, 절묘한 라인 유지
```

#### 5단계: 반전
**현재**: 완전히 다른 이유/목적/관계 공개 - 설명 없이 한 문장으로 끝

**개선안**:
```
5. 반전 (Twist Reveal)
   - 규칙: 딱 1문장, 설명 금지, 추가 해설 금지
   - 형식: 대사 OR 행동 묘사
   - 타이밍: 이 문장 직후 즉시 종료
   - 예시:
     * ✅ "알고 보니 골프공이 벙커 깊숙이 박혀서 빼는 거였대요"
     * ❌ "알고 보니... 였어요. 그래서 우리는 웃으면서..."
   - 반전 후 금지사항:
     * 리액션 추가 설명 ❌
     * 교훈이나 메시지 ❌
     * "여러분은요?" 같은 CTA ❌
```

## 🎯 추가 제안: 6단계 추가?

**제안**: 반전 직후 **무음 여운** 1문장 추가

```
6. 무음 여운 (Silent Echo) - 선택사항
   - 형식: 극도로 짧은 리액션 (5자 이내)
   - 예시: "...뭐야", "헐", "진짜?", "아..."
   - 효과: 시청자와 동일한 반응 → 공감대 형성
   - 주의: 이것도 설명하면 안 됨
```

**Mama-nim 의견**: 이거 넣을까요? 아니면 5단계에서 딱 끝내는 게 더 강력할까요?
