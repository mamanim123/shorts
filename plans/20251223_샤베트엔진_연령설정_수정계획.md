# 샤베트 엔진 연령 설정 문제 수정 계획

**작성일**: 2025-12-23  
**문제**: 샤베트 의상,헤어 엔진 + 50대 선택 시 연령이 대본에 반영되지 않음  
**근본 원인**: 커스텀 엔진이 V3 엔진 로직을 공유하면서도 연령 변수 치환이 누락됨

---

## 🔍 문제 분석

### 발견된 문제점

1. **연령 설정이 반영되지 않음**: 50대를 선택했으나 대본에 연령이 전혀 반영되지 않음
2. **샤베트 엔진의 특성이 누락됨**: 샤베트 엔진의 "상큼·발랄·엉뚱·귀여운" 톤이 아닌 일반적인 톤으로 생성됨
3. **V3 엔진 로직 공유 문제**: 커스텀 엔진이 `isV3Family || isCustomEngine` 조건으로 V3 로직을 공유하면서도 일부 기능만 누락

### 생성된 대본 분석

```
오늘 여기서 내 마음 다 보여줄게, 준비됐어?
눈꽃이 흩날리는 필드에서 혜진이가 수줍게 웃더라고.
하얀 눈보다 더 하얀 니트를 입고 서 있는데 정말 눈이 부셨어.
...
```

**문제점**:
- ❌ 연령 언급 없음 (50대 설정이 무시됨)
- ❌ 샤베트 톤 아님 (일반적인 로맨스 톤)
- ❌ 슬롯 기반 헤어 스타일 미적용
- ❌ 샤베트 특유의 "반전 코미디" 구조 없음

---

## 🔎 근본 원인 (Root Cause)

### 1. 코드 구조 문제

**`geminiService.ts` Line 1101**:
```typescript
} else if (isV3Family || isCustomEngine) {
  // 커스텀 엔진도 V3 로직을 공유함
  const v3Vars = isChatGPT ? generateSafeV3Variables(input.category) : generateV3Variables(input.category);
  // ...
}
```

**문제**:
- 커스텀 엔진이 V3 엔진과 같은 블록에서 처리됨
- `v3Vars` (V3 의상 변수)를 생성하지만, 커스텀 엔진 프롬프트에는 주입하지 않음
- 연령 변수(`ageTokens`)도 생성하지만 커스텀 엔진 프롬프트에 치환하지 않음

### 2. 연령 데이터 흐름

```
[UI - ConfigPanel.tsx Line 272]
  ↓ 사용자가 "50대" 선택
[UserInput.targetAge = "50대"]
  ↓
[geminiService.ts Line 858]
  ↓ const ageTokens = buildAgeTokens(input.targetAge);
  ↓ ageTokens = { digits: "50", korean: "50대", englishShort: "in her 50s" }
[분기점 - Line 1101]
  ├─ [V3 엔진] → userPrompt에 ageTokens 주입 ✅
  │   └─ Line 1231: Target Age: ${input.targetAge}
  │
  └─ [커스텀 엔진] → systemPrompt만 설정, 변수 치환 없음 ❌
      └─ Line 1108: systemPrompt = selectedEnginePrompt
```

### 3. 샤베트 엔진 프롬프트의 하드코딩

`engine_config.json`의 샤베트 엔진들:
- `CUSTOM_1765968277784`
- `CUSTOM_1765973385899`
- `CUSTOM_1766109481051`
- `CUSTOM_1766400278267`

**프롬프트 내용**:
```
모든 여성 캐릭터는 시스템에서 설정된 나이, 관리 잘된 글래머러스 실루엣...
stunning Korean woman in her [시스템에서 설정된 나이]
```

**문제**: 플레이스홀더만 있고 실제 값으로 치환되지 않음

## 💡 해결 방안

### Option A: 커스텀 엔진 전용 경로 분리 (권장) ⭐

**핵심 아이디어**:
- 커스텀 엔진을 V3 로직에서 완전히 분리
- 커스텀 엔진 전용 변수 치환 로직 추가
- 각 엔진이 독립적으로 작동하도록 보장

**장점**:
- 커스텀 엔진의 독립성 보장
- V3 엔진과의 간섭 제거
- 향후 커스텀 엔진 확장 용이
- 디버깅 및 유지보수 쉬움

**구현 방법**:

#### 1. `engine_config.json` 수정

4개의 샤베트 엔진 프롬프트에서 플레이스홀더 추가:

**변경 전**:
```
모든 여성 캐릭터는 시스템에서 설정된 나이, 관리 잘된 글래머러스 실루엣...
stunning Korean woman in her [시스템에서 설정된 나이]
```

**변경 후**:
```
모든 여성 캐릭터는 {{TARGET_AGE}}, 관리 잘된 글래머러스 실루엣...
stunning Korean woman {{TARGET_AGE_EN}}
```

#### 2. `geminiService.ts` 수정

**Line 1101-1130 부근 수정**:

```typescript
// 변경 전
} else if (isV3Family || isCustomEngine) {
  const isChatGPT = input.targetService === 'CHATGPT';
  
  if (isEngineOff) {
    systemPrompt = "";

    systemPrompt = selectedEnginePrompt;

} else if (isCustomEngine) {
  // === 커스텀 엔진 전용 경로 (V3와 완전 분리) ===
  const isChatGPT = input.targetService === 'CHATGPT';
  
  // 1. 연령 토큰 생성
  const customAgeTokens = buildAgeTokens(input.targetAge);
  
  // 2. 의상 변수 생성 (커스텀 엔진도 필요한 경우)
  const customVars = isChatGPT ? generateSafeV3Variables(input.category) : generateV3Variables(input.category);
  
  // 3. 프롬프트 변수 치환
  systemPrompt = selectedEnginePrompt
    .replace(/\{\{TARGET_AGE\}\}/g, customAgeTokens.korean)
    .replace(/\{\{TARGET_AGE_EN\}\}/g, customAgeTokens.englishShort)
    .replace(/\[시스템에서 설정된 나이\]/g, customAgeTokens.digits + 's')
    .replace(/시스템에서 설정된 나이/g, customAgeTokens.korean);
  
  console.log(`[Engine] Custom engine "${input.engineVersion}" loaded with age: ${customAgeTokens.korean}`);
  
  // 4. userPrompt 구성 (커스텀 엔진용 간소화)
  userPrompt = `
Topic: ${resolvedTopic}
Outfit Style: ${input.category || "0. 선택안함"}
Target Age: ${customAgeTokens.korean}
Additional Context: ${topicContext || "None"}

[CRITICAL]
- Follow the system prompt rules strictly.
- Use the age setting (${customAgeTokens.korean}) for all characters.
- Generate valid JSON output only.
  `;

} else if (isV3Family) {
  // === V3 엔진 전용 경로 (기존 로직 유지) ===
  const isChatGPT = input.targetService === 'CHATGPT';
  
  if (isEngineOff) {
    systemPrompt = "";
  } else if (input.useViralMode && input.scenarioMode === ScenarioMode.DOUBLE_ENTENDRE) {
    systemPrompt = SYSTEM_PROMPT_VIRAL;
  } else if (isChatGPT) {
    systemPrompt = SYSTEM_PROMPT_CHATGPT;
  } else if (input.targetService === 'CLAUDE') {
    systemPrompt = SYSTEM_PROMPT_CLAUDE;
  } else {
    systemPrompt = selectedEnginePrompt;
  }
  
  const v3Vars = isChatGPT ? generateSafeV3Variables(input.category) : generateV3Variables(input.category);
  // ... (기존 V3 로직 계속)
}
```

### Option B: 조건부 변수 치환 (간단하지만 비권장)

V3와 커스텀 엔진을 계속 공유하되, 커스텀 엔진일 때만 추가 치환 수행.
