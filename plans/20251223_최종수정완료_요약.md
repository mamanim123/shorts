## 🎯 핵심 수정사항

### 수정 1: 후처리 OFF면 호출 안 함
```typescript
// Before
const processedScripts = postProcessScripts(...); // 항상 호출

// After
const shouldPostProcess = autoEnhanceOnGeneration ?? false;
const processedScripts = shouldPostProcess 
  ? postProcessScripts(...) 
  : result.scripts; // OFF면 LLM 출력 그대로
```

### 수정 2: 기본값 변경
```typescript
// Before
autoEnhanceOnGeneration ?? true  // 위험!

// After
autoEnhanceOnGeneration ?? false // 안전!
```
