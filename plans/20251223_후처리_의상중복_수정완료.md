## 🔍 문제 분석

### 증상
- 이미지 프롬프트에 의상이 중복으로 추가됨
- 예: `"Navy Lace Mini Skirt + Red Micro Mini Skirt + White Chiffon Shirt + Red Micro Mini Skirt"`

### 원인

#### 1. LLM은 정상적으로 의상 생성
- ✅ LLM이 받은대본.txt에서 확인한 결과, 의상을 정확히 한 벌씩만 생성
- ✅ 예: `"Outfit: White Halter-neck Knit + Red Micro Mini Skirt"`

#### 2. 후처리에서 의상 추가
**위치**: `enhanceScenePrompt` 함수 (Line 318-367)

**문제점**:
1. **autoEnhance 체크 없음**: 후처리 OFF여도 의상 추가
2. **LLM 의상 체크 실패**: LLM이 이미 의상을 포함했는데도 추가
3. **다른 의상 체크 실패**: 
   - LLM: `"Outfit: White Halter-neck Knit"`
   - 후처리: `"Outfit: Navy Lace Mini Skirt"` 추가 시도
   - 체크: `!updated.includes("Outfit: Navy Lace Mini Skirt")` → `true`
   - 결과: 중복 추가!

## 💡 해결 방법

### 수정 1: LLM이 의상 제공했는지 체크
```typescript
// [FIX] Check if LLM already provided outfit
const llmProvidedOutfit = updated.includes("Outfit:");
if (llmProvidedOutfit) {
  hasAddedCharacterOutfit = true; // 이미 있음으로 마킹
}
```

### 수정 2: autoEnhance가 false면 의상 추가 안 함
```typescript
// [FIX] Only inject if autoEnhance is true AND LLM didn't already provide outfit
if (options.autoEnhance && !llmProvidedOutfit && options.femaleOutfit && !hasAddedCharacterOutfit) {
  // 의상 추가
}
```

### 수정 3: 모든 의상 추가 로직에 동일 체크 적용
- 캐릭터별 의상 (Line 323-341)
- 글로벌 여성 의상 (Line 346-351)

## ✅ 수정 효과

### Before (수정 전)
```
후처리 OFF여도 의상 추가됨
→ "Outfit: White Halter + Red Skirt, Outfit: Navy Lace Skirt"
```

### After (수정 후)
```
1. LLM이 의상 포함 → 후처리에서 추가 안 함
2. 후처리 OFF → 의상 추가 안 함
3. 후처리 ON + LLM이 의상 없음 → 의상 추가
```

## 🧪 테스트 방법

### 테스트 1: 후처리 OFF + LLM 의상 있음
1. 후처리 설정 OFF
2. 샤베트 엔진으로 대본 생성
3. **기대 결과**: LLM이 생성한 의상만 있음, 중복 없음

### 테스트 2: 후처리 ON + LLM 의상 있음
1. 후처리 설정 ON
2. 샤베트 엔진으로 대본 생성
3. **기대 결과**: LLM이 생성한 의상만 있음, 후처리가 추가 안 함

### 테스트 3: 후처리 ON + LLM 의상 없음
1. 후처리 설정 ON
2. 일반 엔진으로 대본 생성 (의상 미포함)
3. **기대 결과**: 후처리가 의상 추가
