## 🐛 발견된 버그

### 문제 상황
마마님께서 발견하신 프롬프트:
```
Slot B — narrator, stunning 40대 Korean woman [short bob hair, petite but glamorous figure, 
outfit: Beige Halter-neck Knit + Navy Tight Mini Skirt (Elegant)] 
looking directly at the camera with an awkward but cute smile, 
snowy golf course in the background.
```

**❌ 악세서리가 없음!**
- earrings 없음
- necklace 없음
- watch 없음

---

## 🔍 원인 분석

### Before (버그 코드)
```typescript
const hasFemaleCharacter = characterIds.some((id: string) => 
  id === 'A' || id === 'C' || /Slot [AC]/i.test(id)  // ← B가 빠짐!
);
```

**문제점:**
- Slot A와 Slot C만 여성 캐릭터로 인식
- **Slot B는 체크하지 않음**
- 결과: Slot B 캐릭터에게 악세서리가 추가되지 않음

---

## ✅ 수정 내용

### After (수정 코드)
```typescript
const hasFemaleCharacter = characterIds.some((id: string) => 
  id === 'A' || id === 'B' || id === 'C' || /Slot [ABC]/i.test(id)  // ← B 추가!
);

if (hasFemaleCharacter) {
  const mainCharacterId = characterIds.find((id: string) => 
    id === 'A' || id === 'B' || id === 'C' || /Slot [ABC]/i.test(id)  // ← B 추가!
  ) || 'A';
  
  // 악세서리 추가 로직...
}
```

**수정 사항:**
- ✅ Slot B를 여성 캐릭터 체크에 추가
- ✅ mainCharacterId 찾기 로직에도 B 추가
- ✅ 정규식 패턴을 `[AC]`에서 `[ABC]`로 변경
