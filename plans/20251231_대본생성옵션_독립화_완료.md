# 대본 생성 옵션 독립화 완료

**작성일**: 2025-12-31  
**작업명**: 대본 생성 옵션과 생성 모드 분리  
**상태**: ✅ 완료

---

## 🎯 작업 목표

### 문제 상황
- "대본 생성 옵션"에서 "대본+이미지" 선택 시
- "생성 모드"의 "대본→이미지" 버튼도 같이 활성화됨
- 두 시스템이 `input.customScript` 필드를 공유하여 충돌 발생

### 해결 방안
- **새로운 필드 추가**: `shortsGenerationMode`
- "대본 생성 옵션"은 쇼츠 생성기 전용으로만 사용
- "생성 모드" 4개 버튼과 완전히 독립적으로 동작

---

## ✅ 변경 사항

### 1️⃣ **types.ts 수정**

**Before:**
```typescript
export interface UserInput {
  customScript?: string; // 생성 모드와 공유
  // ...
}
```

**After:**
```typescript
export interface UserInput {
  customScript?: string; // 생성 모드 전용
  shortsGenerationMode?: 'script-only' | 'script-image'; // 쇼츠 생성기 전용
  // ...
}
```

---

### 2️⃣ **ConfigPanel.tsx 수정**

**Before (충돌 발생):**
```typescript

  대본만
</button>
<button onClick={() => onChange({ ...input, customScript: "" })}>
  대본+이미지
</button>
```

**After (독립 동작):**
```typescript
<button onClick={() => onChange({ ...input, shortsGenerationMode: 'script-only' })}>
  대본만
</button>
<button onClick={() => onChange({ ...input, shortsGenerationMode: 'script-image' })}>
  대본+이미지
</button>
```

### 3️⃣ **App.tsx 초기값 설정**


## 📋 필드 역할 분리

### `customScript` (생성 모드 전용)
- **사용처**: 생성 모드 4개 버튼
  - 스토리생성: `undefined`
  - 대본→이미지: `""`
  - JSON입력: `undefined`
  - 대본분석: `undefined`
- **영향**: 왼쪽 패널 하단 버튼 표시

### `shortsGenerationMode` (쇼츠 생성기 전용)
- **사용처**: 대본 생성 옵션 버튼
  - 대본만: `'script-only'`
  - 대본+이미지: `'script-image'`
- **영향**: 쇼츠 생성기 동작 모드

## 🔍 작동 원리

### 대본 생성 옵션 (쇼츠 생성기 전용)
```typescript
// 대본만
onClick={() => onChange({ 
  ...input, 
  shortsGenerationMode: 'script-only' 
})}

// 대본+이미지
onClick={() => onChange({ 
  ...input, 
  shortsGenerationMode: 'script-image' 
})}
```

**특징:**
- `customScript` 필드에 영향 없음
- 생성 모드 버튼과 독립적
- 쇼츠 생성기에서만 참조

### 생성 모드 (기존 시스템)
```typescript
// 스토리생성
onClick={() => { 
  onChange({ ...input, customScript: undefined, customJson: undefined }); 
  onOpenShortform(); 
}}

// 대본→이미지
onClick={() => { 
  onChange({ ...input, customScript: "", customJson: undefined }); 
  onOpenShortform(); 
}}
```

**특징:**
- `shortsGenerationMode` 필드에 영향 없음
- 대본 생성 옵션과 독립적
- 기존 로직 그대로 유지
