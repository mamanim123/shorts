# 왼쪽 패널 UI 개선 완료 (수정본)

**작성일**: 2025-12-31  
**작업명**: 왼쪽 패널 스토리 엔진 삭제 및 생성모드/장르선택 추가  
**상태**: ✅ 완료

---

## 🎯 작업 목표

### 문제 상황
1. 대본 만드는 기능이 **두 군데**에 존재:
   - 메인 화면 왼쪽 패널 - "스토리 생성하기"
   - 쇼츠 생성기 - "대본 만들기"
2. 왼쪽 패널의 "스토리 엔진" 선택이 불필요함
3. **장르 선택이 잘못됨**: 의상 스타일(OUTFIT_STYLES)을 장르로 사용 중
4. **세로 배치로 공간 낭비**: 장르선택과 타겟연령이 각각 별도 섹션

### 해결 방안
1. **"스토리 엔진 선택" 섹션 삭제**
2. **새로 추가:**
   - 생성모드 (대본만 / 대본+이미지)
   - **진짜 장르 선택** (쇼츠 생성기의 장르 가져오기)
3. **가로 배치로 공간 절약**: 장르선택 + 타겟연령을 한 섹션에 가로 배치

---

## ✅ 변경 사항

### 1️⃣ **삭제됨**
- ❌ **"스토리 엔진 선택"** 섹션 전체 제거
- ❌ **잘못된 장르 선택** (OUTFIT_STYLES 사용) 제거

### 2️⃣ **새로 추가됨**

#### A. 생성모드 (대본만 / 대본+이미지)
```tsx
<div className="grid grid-cols-2 gap-2">
  <button>대본만</button>
  <button>대본+이미지</button>
</div>
```

**기능:**
- **대본만**: `customScript: undefined` → 대본만 생성
- **대본+이미지**: `customScript: ""` → 대본 + 이미지 프롬프트 생성

#### B. 장르 선택 + 타겟 연령 (가로 배치) ✨ NEW
```tsx
<div className="grid grid-cols-2 gap-3">
  {/* 장르 선택 */}
  <div>
    <select value={input.scenarioMode}>
      <option value="affair-suspicion">불륜 의심</option>
      <option value="office-romance">직장 로맨스</option>
      ...
    </select>
  </div>

  {/* 타겟 연령 */}
  <div>
    <select value={input.targetAge}>
      <option>20대</option>
      <option>30대</option>
      ...
    </select>
  </div>
</div>
```

**장르 목록 (scenarioMode):**
- 불륜 의심 (Affair Suspicion)
- 직장 로맨스 (Office Romance)
- 나이차 로맨스 (Age Gap)
- 금지된 사랑 (Forbidden Love)
- 재회 (Reunion)
- 비밀 짝사랑 (Secret Admirer)

**연령 목록:**
- 20대, 30대, 40대, 50대, 60대

## 🎨 UI 개선 사항

### 1. 장르 선택 수정
**Before:**
```tsx
// 잘못된 장르 (의상 스타일)
<select value={input.category}>
  {OUTFIT_STYLES.map((style) => (
    <option>{style}</option>  // "0. 선택안함", "1. 모던 시크" 등
  ))}
</select>
```

**After:**
```tsx
// 올바른 장르 (시나리오 모드)
<select value={input.scenarioMode}>
  <option value="affair-suspicion">불륜 의심</option>
  <option value="office-romance">직장 로맨스</option>
  <option value="age-gap">나이차 로맨스</option>
  ...
</select>
```

┌─────────────────────┐
│ 타겟 연령           │
│ [드롭다운]          │

**After (가로 배치):**
```

```

## 🔍 작동 원리

### 1. 생성모드 선택
```typescript
// 대본만
onClick={() => onChange({ ...input, customScript: undefined })}

// 대본+이미지
onClick={() => onChange({ ...input, customScript: "" })}
```

### 2. 장르 선택 (scenarioMode)
```typescript
<select
  value={input.scenarioMode || 'affair-suspicion'}
  onChange={(e) => onChange({ ...input, scenarioMode: e.target.value as any })}
>
  <option value="affair-suspicion">불륜 의심 (Affair Suspicion)</option>
  <option value="office-romance">직장 로맨스 (Office Romance)</option>
  ...
</select>
```

### 3. 타겟 연령
```typescript
<select
  value={input.targetAge || "40대"}
  onChange={(e) => onChange({ ...input, targetAge: e.target.value })}
>
  {AGE_GROUPS.map((age) => (
    <option key={age} value={age}>{age}</option>
  ))}
</select>
```
