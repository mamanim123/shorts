**작성일**: 2025-12-31  
**작업명**: 이미지 프롬프트 강화 (악세서리 + 동작 + 여러 명 등장)  
**목표**:

## 📋 현재 문제점

### 문제 1: 악세서리가 나오지 않음
**증상**: 
```
Slot B — narrator, stunning 40대 Korean woman [...] 
(악세서리 없음) ❌
```

**원인**: 
- `ensureAccessory` 함수가 "Outfit:" 패턴을 찾지 못함
- AI가 복잡한 프롬프트를 생성하면 삽입 위치를 못 찾음

### 문제 2: 동작이 너무 약함
**증상**:
```
standing on a snowy golf course  ← 너무 정적!
walking gracefully  ← 약함!
standing still  ← 지루함!
```

**원인**:
- `ensureDynamicAction` 함수가 "standing", "walking" 같은 약한 동작을 감지하면 건너뜀
- AI가 약한 동작을 넣으면 제 함수가 "이미 동작이 있네?"하고 패스

### 문제 3: 여러 명이 함께 나오지 않음
**증상**:
```
Scene 1: Slot B만 (혼자)
Scene 2: Slot B + 남성 (2명)
Scene 3: Slot B + 남성 (2명)
Scene 4: Slot B만 (혼자)  ← 대부분 혼자!
```

**원인**:
- AI가 "단체샷 필수" 지시를 무시함
- 후처리 검증 로직이 없음

## 🎯 해결 방안

### 해결 1: 악세서리 추가 로직 강화

#### 현재 코드 (문제)
```typescript
const ensureAccessory = (prompt: string, characterId: string): string => {
  // "Outfit:" 패턴을 찾아서 뒤에 추가
  let result = prompt.replace(
    /(Outfit:[^,]+),/i,
    `$1, wearing ${accessory},`
  );
  
  // 못 찾으면 "Korean woman" 뒤에 추가
  if (result === prompt) {
    result = prompt.replace(
      /(Korean woman in (?:her|his) \d+s),/i,
      `$1, wearing ${accessory},`
    );
  }
  
  return result;
};
```

#### 개선 코드
```typescript
const ensureAccessory = (prompt: string, characterId: string): string => {
  if (!prompt) return prompt;
  
  const accessories: Record<string, string> = {
    A: "diamond stud earrings + gold luxury watch",
    B: "pearl drop earrings + silver pendant necklace",
    C: "simple platinum hoops + black leather strap watch"
  };
  
  // 이미 악세서리가 있는지 확인
  const hasAccessory = /earrings|necklace|bracelet|watch|ring|accessory|jewelry|pendant|brooch|hairpin|scarf/i.test(prompt);
  
  if (hasAccessory) {
    return prompt;
  }
  
  const slotMatch = characterId.match(/Slot ([ABC])/i) || characterId.match(/^([ABC])$/);
  const slot = slotMatch ? slotMatch[1].toUpperCase() : 'A';
  const accessory = accessories[slot] || accessories.A;
  
  // 1순위: "outfit:" 뒤에 추가
  let result = prompt.replace(
    /(outfit:[^,]+),/i,
    `$1, wearing ${accessory},`
  );
  
  // 2순위: "], " 패턴 뒤에 추가 (Slot 정의 끝)
  if (result === prompt) {
    result = prompt.replace(
      /(\[short bob hair[^\]]*\]),/i,
      `$1, wearing ${accessory},`
    );
  }
  
  // 3순위: "Korean woman in her 40s" 뒤에 추가
  if (result === prompt) {
    result = prompt.replace(
      /(Korean woman in (?:her|his) \d+s),/i,
      `$1, wearing ${accessory},`
    );
  }
  
  // 4순위: 첫 번째 쉼표 뒤에 강제 삽입
  if (result === prompt && prompt.includes(',')) {
    const firstCommaIndex = prompt.indexOf(',');
    result = prompt.slice(0, firstCommaIndex + 1) + ` wearing ${accessory},` + prompt.slice(firstCommaIndex + 1);
  }
  
  return result.replace(/,\s*,/g, ', ').trim();
};
```

### 해결 2: 약한 동작을 강한 동작으로 교체

#### 현재 코드 (문제)
```typescript
const hasAction = /leaning|crossing|gesturing|tilting|stepping|reaching|turning|laughing|walking|standing|sitting|running|jumping|dancing|pointing|waving|nodding|shaking|bending|stretching/i.test(prompt);

if (hasAction) {
  return prompt;  // ← "standing", "walking" 있으면 그냥 리턴!
}
```

#### 개선 코드
```typescript
const ensureDynamicAction = (prompt: string, sceneNumber: number): string => {
  if (!prompt) return prompt;
  
  const actions = [
    "leaning forward with curiosity",
    "crossing arms defensively",
    "gesturing expressively with hands",
    "tilting head in confusion",
    "stepping back in surprise",
    "reaching out tentatively",
    "turning away in frustration",
    "laughing while covering mouth"
  ];
  
  // 약한 동작 목록
  const weakActions = /\b(standing still|standing|walking gracefully|walking|sitting quietly|sitting|looking)\b/gi;
  
  // 강한 동작이 있는지 확인
  const hasStrongAction = /leaning|crossing|gesturing|tilting|stepping|reaching|turning|laughing|running|jumping|dancing|pointing|waving|nodding|shaking|bending|stretching/i.test(prompt);
  
  if (hasStrongAction) {
    return prompt; // 이미 강한 동작이 있으면 그대로
  }
  
  const selectedAction = actions[sceneNumber % actions.length];
  
  // 약한 동작을 강한 동작으로 교체
  if (weakActions.test(prompt)) {
    return prompt.replace(weakActions, selectedAction).replace(/,\s*,/g, ', ').trim();
  }
  
  // 약한 동작도 없으면 추가
  let result = prompt.replace(
    /(Korean (?:woman|man) in (?:her|his) \d+s),/i,
    `$1, ${selectedAction},`
  );
  
  if (result === prompt) {
    result = `${selectedAction}, ${prompt}`;
  }
  
  return result.replace(/,\s*,/g, ', ').trim();
};
```

### 해결 3: 여러 명 등장 로직 (하이브리드)

#### Step 1: AI 프롬프트 강화 (엔진 설정)
- 샤베트 엔진 프롬프트에 더 강력한 지시 추가
- 이 부분은 나중에 별도 작업

#### Step 2: 후처리 검증 + 자동 수정
```typescript
const enforceMultipleCharacters = (scenes: Scene[]): Scene[] => {
  if (!scenes || scenes.length === 0) return scenes;
  
  // 1. 현재 여러 명 등장 비율 계산
  const multiCount = scenes.filter(s => 
    s.characterIds && s.characterIds.length >= 2
  ).length;
  
  const ratio = multiCount / scenes.length;
  
  // 2. 비율이 30% 미만이면 강제 수정
  if (ratio < 0.3) {
    console.log(`[Multiple Characters] 현재 비율: ${(ratio * 100).toFixed(1)}% → 30% 이상으로 강제 조정`);
    
    // 짝수 씬에 Slot B 추가 (1, 3, 5, 7번 씬)
    const targetIndices = [1, 3, 5, 7];
    
    targetIndices.forEach(idx => {
      if (scenes[idx] && scenes[idx].characterIds) {
        const currentIds = scenes[idx].characterIds;
        
        // Slot A만 있으면 Slot B 추가
        if (currentIds.length === 1 && (currentIds[0] === 'A' || /Slot A/i.test(currentIds[0]))) {
          scenes[idx].characterIds.push('B');
          
          // longPrompt에 Slot B 추가
          const slotBDesc = "Slot B — stunning Korean woman in her 40s [short bob hair, petite but glamorous figure], wearing pearl drop earrings + silver pendant necklace, standing beside her with a curious expression";
          
          scenes[idx].longPrompt = scenes[idx].longPrompt.replace(
            /(Slot A[^.]+\.)/,
            `$1 ${slotBDesc}.`
          );
          
          // shortPrompt에도 추가
          if (scenes[idx].shortPrompt) {
            scenes[idx].shortPrompt = scenes[idx].shortPrompt.replace(
              /(Slot A[^,]+),/,
              '$1, and Slot B [short bob hair],'
            );
          }
        }
      }
    });
  }
  
  return scenes;
};
```
