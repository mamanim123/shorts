### 2. ✅ 동작 로직 강화 (약한 동작 교체)

**수정 파일**: `services/geminiService.ts` (라인 868-925)

**개선 내용**:
```typescript
// Before: 동작이 있으면 건너뜀
if (hasAction) {
  return prompt;  // "standing", "walking" 있으면 그냥 리턴
}

// After: 약한 동작을 강한 동작으로 교체
const weakActions = /\b(standing still|standing|walking gracefully|walking|sitting quietly|sitting|looking at|looking)\b/gi;

if (weakActions.test(prompt)) {
  return prompt.replace(weakActions, selectedAction);  // 교체!
}
```

**효과**:
- ✅ "standing" → "leaning forward with curiosity"
- ✅ "walking gracefully" → "crossing arms defensively"
- ✅ 모든 씬이 역동적인 동작으로 변경

### 3. ✅ 여러 명 등장 로직 추가 (하이브리드)

```typescript
const enforceMultipleCharacters = (scenes: any[]): any[] => {
  // 1. 현재 여러 명 등장 비율 계산
  const multiCount = scenes.filter(s => 
    s.characterIds && s.characterIds.length >= 2
  ).length;
  
  const ratio = multiCount / scenes.length;
  
  // 2. 비율이 30% 미만이면 강제 수정
  if (ratio < 0.3) {
    // 짝수 씬에 Slot B 추가 (1, 3, 5, 7번 씬)
    targetIndices.forEach(idx => {
      scenes[idx].characterIds.push('B');
      // longPrompt와 shortPrompt에 Slot B 추가
    });
  }
  
  return scenes;
};
```

**적용 위치**:
- ✅ Script-to-Image 모드 (라인 1335-1338)
- ✅ Main Story 모드 (라인 1719-1722)

**효과**:
- ✅ 전체 씬 중 최소 30% 이상은 2명 이상 등장
- ✅ AI가 혼자만 등장시켜도 자동으로 여러 명 추가
