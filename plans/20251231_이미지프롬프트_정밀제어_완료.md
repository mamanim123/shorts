# 이미지 프롬프트 정밀 제어 및 규칙 강제화 완료

**작성일**: 2025-12-31  
**작업명**: 이미지 프롬프트 씬 번호 추가, 텍스트 제거 및 규칙 강제화  
**상태**: ✅ 완료

---

## 🎯 작업 목표

### 문제 상황
- AI가 생성한 이미지 프롬프트가 대본의 상황(구도, 상호작용)을 제대로 반영하지 못함
- 필수 액세서리(시계, 목걸이) 누락
- 이미지 내에 불필요한 텍스트가 포함됨
- 씬 번호가 없어 관리가 어려움

### 해결 방안
- **프롬프트 템플릿 개조**: AI에게 더 강력한 지침과 예시 제공
- **후처리 로직(코드) 강화**: AI가 실수해도 코드가 자동으로 보정하도록 구현
- **씬 번호 및 텍스트 방지**: 모든 프롬프트에 씬 번호와 텍스트 제거 태그 강제 삽입

---

## ✅ 변경 사항

### 1️⃣ **씬 번호(Scene Number) 강제화**
- 모든 이미지 프롬프트 맨 앞에 `Scene [번호]. ` 형식이 붙도록 수정되었습니다.
- AI가 생성할 때도 이 형식을 지키도록 템플릿을 수정했으며, 코드가 후처리 과정에서 번호가 없거나 틀리면 자동으로 교정합니다.

### 2️⃣ **이미지 내 텍스트 제거 (No Text)**
- 이미지 생성 시 글자가 들어가지 않도록 모든 프롬프트 끝에 다음 태그를 자동으로 추가합니다.
- `no text, no letters, no typography, no watermarks, no words`

### 3️⃣ **구도(Shot Type) 및 상호작용 강제**
- 2인 이상 등장하는 씬에서 구도 명시(`Two-shot`, `Three-shot`)가 누락된 경우, 인물 수에 따라 자동으로 맨 앞에 삽입합니다.
- 2인 이상 씬에는 반드시 인물 간의 연결(`eye contact`, `interaction`)이 포함되도록 보정합니다.

### 4️⃣ **슬롯별 필수 액세서리 고정**
- **Slot A**: `wearing a luxury diamond watch` (럭셔리 다이아몬드 시계)
- **Slot B**: `wearing a simple silver necklace` (심플 실버 목걸이)
- **Slot C**: `wearing a gold hoop earrings` (골드 후프 귀걸이)
- 프롬프트에 슬롯 이름은 있으나 액세서리 묘사가 없는 경우 코드가 자동으로 주입합니다.

## 🔍 작동 원리 (후처리 로직)

```typescript
// 1. 씬 번호 강제 (맨 앞)
updated = `Scene ${sceneNumber}. ` + updated;

// 2. 구도 강제 (씬 번호 바로 뒤)
if (인원 >= 2) updated = updated.replace(/Scene \d+\. /, "$&Two-shot, ");

// 3. 액세서리 주입
if (SlotA_있음 && 시계_없음) updated = updated.replace("Slot A", "Slot A [wearing a luxury diamond watch]");

// 4. 텍스트 방지 (맨 뒤)
updated += ", no text, no letters...";
```
