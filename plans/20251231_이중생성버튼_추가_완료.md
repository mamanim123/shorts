**작성일**: 2025-12-31

---

## 🎯 작업 목표

### 문제 상황
- 왼쪽 패널의 "스토리 생성하기" 버튼은 `geminiService.ts` 사용
- 쇼츠 생성기는 자체 로직 사용
- **두 시스템이 완전히 다름**

### 해결 방안
- 왼쪽 패널 하단에 **두 개의 버튼** 추가
  1. **스토리 생성**: 기존 `geminiService.ts` 로직
  2. **쇼츠 대본 생성**: 쇼츠 생성기 탭으로 이동

---

## ✅ 변경 사항

### Before (수정 전)
```tsx
<Button onClick={handleGenerate}>
  스토리 생성하기
</Button>
```

### After (수정 후)
```tsx
<div className="grid grid-cols-2 gap-3">
  {/* 1. 스토리 생성 (기존 로직) */}
  <Button
    onClick={handleGenerate}
    className="bg-purple-600"
  >
    <Wand2 />
    스토리 생성
  </Button>

  {/* 2. 쇼츠 대본 생성 (쇼츠 생성기로 이동) */}
  <Button
    onClick={() => {
      setRightTab('shorts-generator');
      setSidebarTab('history');
      setHistorySubTab('stories');
    }}
    className="bg-pink-600"
  >
    <Sparkles />
    쇼츠 대본 생성
  </Button>
</div>
```

## 📋 버튼 기능

### 1️⃣ **스토리 생성** (보라색)
- **아이콘**: Wand2 (마법봉)
- **색상**: Purple (보라색)
- **기능**: 
  - 기존 `handleGenerate()` 함수 호출
  - `geminiService.ts`의 `generateStory()` 사용
  - 템플릿 시스템 사용
  - 결과를 "숏폼 결과" 탭에 표시

### 2️⃣ **쇼츠 대본 생성** (분홍색)
- **아이콘**: Sparkles (반짝임)
- **색상**: Pink (분홍색)
- **기능**:
  - 쇼츠 생성기 탭으로 이동
  - 보관함 탭 활성화
  - 생성 대본 서브탭 선택
  - 쇼츠 생성기의 자체 로직 사용

## 🔍 작동 원리

### 스토리 생성 버튼
```typescript
<Button
  onClick={handleGenerate}  // 기존 함수 호출
  isLoading={loading}
  className="bg-purple-600"
>
  스토리 생성
</Button>
```

**실행 흐름:**
1. `handleGenerate()` 호출
2. `generateStory(input, signal, activeTemplate)` 실행
3. `geminiService.ts` 로직 사용
4. 결과를 "숏폼 결과" 탭에 표시

### 쇼츠 대본 생성 버튼
```typescript
<Button
  onClick={() => {
    setRightTab('shorts-generator');    // 쇼츠 생성기 탭으로
    setSidebarTab('history');           // 보관함 탭 활성화
    setHistorySubTab('stories');        // 생성 대본 서브탭
  }}
  className="bg-pink-600"
>
  쇼츠 대본 생성
</Button>
```

**실행 흐름:**
1. 우측 탭을 "쇼츠 생성기"로 변경
2. 좌측 탭을 "보관함"으로 변경
3. 서브탭을 "생성 대본"으로 변경
4. 쇼츠 생성기 UI 표시

## 📊 두 시스템 비교

| 항목 | 스토리 생성 | 쇼츠 대본 생성 |
|------|------------|---------------|
| **생성 함수** | `generateStory()` (geminiService.ts) | 쇼츠 생성기 자체 로직 |
| **장르** | `input.scenarioMode` | `genreManager` |
| **템플릿** | `activeTemplate` | `MODE_TEMPLATE` |
| **의상** | `input.category` (OUTFIT_STYLES) | `SHORTFORM_OUTFIT_LIST` |
| **엔진** | `input.engineVersion` | 자체 엔진 |
| **결과 표시** | 숏폼 결과 탭 | 쇼츠 생성기 탭 |

1. 장르, 타겟 연령 선택

3. 로딩 표시 확인
