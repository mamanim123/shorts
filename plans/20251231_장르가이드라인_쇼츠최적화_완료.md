# 쇼츠 최적화 규칙 전면 적용 완료 (장르 가이드라인 포함)

**작성일**: 2025-12-31  
**작업명**: 장르 가이드라인에 쇼츠 최적화 규칙 전면 적용  
**상태**: ✅ 완료

---

## 🎯 작업 목표

### 문제 상황
- `youtube-shorts-script-generator.tsx`만 수정했으나, `services/genreGuidelines.ts`에 여전히 옛날 규칙이 남아있어 AI가 혼란스러워함
- 장르 가이드라인에서 질문형 CTA를 권장하고, 반전 타이밍이 명시되지 않아 규칙 충돌 발생

### 해결 방안
모든 장르 가이드라인(불륜/외도, 친구/동창, 코미디, 로맨스)에 쇼츠 최적화 규칙을 일관되게 적용

---

## ✅ 변경 사항

### 1️⃣ **모든 장르에 훅 강화 규칙 추가**

#### Before
```typescript
1. **훅 (1문장)**: "{{TOPIC}}"과 연관된 의심 상황으로 시작
```

#### After
```typescript
1. **훅 (1문장)**: "{{TOPIC}}"과 연관된 의심 상황으로 **즉시 충격** 시작
   - ❌ 금지: "~한 날이었어", "~에 갔어" (배경 설명)
   - ✅ 필수: "20년 만에 ~", "갑자기 ~", "그 순간 ~" (즉시성 강조)
   - 예: "남편이 새벽 2시에 {{TOPIC}} 관련 문자를 보내고 있더라"
```

---

### 2️⃣ **모든 장르에 반전 타이밍 명시**

#### Before
```typescript
4. **반전 + CTA (각 1문장)**:
   - 의외의 진실 공개
   - 질문형 CTA로 마무리
```

#### After
```typescript
4. **반전 (10~11번째 문장)**:
   - 의외의 진실 공개 (생일 준비, 깜짝 선물, 부모님 병원 등)
   - 또는 진짜 외도였지만 통쾌한 복수
5. **CTA (12번째 문장)**: 감탄형/여운형으로 마무리
   - ❌ 금지: "너라면 어땠을 것 같아?", "~해본 적 있어?" (질문형)
   - ✅ 필수: "이게 뭐야 진짜", "완전 멘붕이었어", "세상에 이런 일이" (감탄형)
```

---

### 3️⃣ **장르별 적용 내역**

| 장르 | 훅 강화 | 반전 타이밍 | CTA 개선 |
|------|---------|-------------|----------|
| 불륜/외도 의심 | ✅ 배경 설명 금지 추가 | ✅ 10~11번째 명시 | ✅ 질문형 금지, 감탄형 강제 |
| 친구/동창 관계 | ✅ 배경 설명 금지 추가 | ✅ 10~11번째 명시 | ✅ 질문형 금지, 감탄형 강제 |
| 코미디/유머 | ✅ 배경 설명 금지 추가 | ✅ 10~11번째 명시 | ✅ 질문형 금지, 감탄형 강제 |
| 로맨스/설렘 | ✅ 배경 설명 금지 추가 | ✅ 10~11번째 명시 | ✅ 질문형 금지, 감탄형 강제 |

---

### 4️⃣ **로맨스/설렘 장르 특별 강화**

마마님께서 제시하신 대본이 로맨스 장르였으므로, 특별히 강화:

```typescript
1. **훅 (1문장)**: "{{TOPIC}}"과 연관된 설렘 상황으로 **즉시 충격** 시작
   - ❌ 금지: "오늘 ~한 날이었어", "아름다운 ~들이" (배경 설명)  // [NEW] 마마님 예시 추가
   - ✅ 필수: "20년 만에 ~", "갑자기 ~", "그 순간 ~" (즉시성 강조)
```

## 📊 Before vs After 비교

### Before (기존 장르 가이드라인)
```
1. 훅: 일반적 시작
2. 배경 설정
3. 전개
4. 반전 + CTA
   - "너희도 이런 경험 있어?" (질문형 CTA 권장)
```

### After (쇼츠 최적화 적용)
```
1. 훅: 즉시 충격 (배경 설명 금지)
2. 배경 설정
3. 전개
4. 반전 (10~11번째 문장)
5. CTA (12번째 문장)
   - "이게 뭐야 진짜" (감탄형 강제)
```

알고 보니 그는 우리 남편의 레슨 프로였어  // ✅ 11번째 반전 완료
이게 뭐야 진짜  // ✅ 감탄형 CTA

## 🎯 최종 효과

이제 **어떤 장르를 선택하든**, **어떤 모드(대본만/대본+이미지)를 선택하든** 모든 대본이 다음을 자동으로 준수합니다:

1. ✅ **훅 강화**: 배경 설명 금지, 즉시 충격으로 시작
2. ✅ **반전 타이밍**: 10~11번째 문장에서 반전 시작
3. ✅ **엔딩 개선**: 질문형 금지, 감탄형/여운형 강제
4. ✅ **템포 가속**: 감정 묘사를 행동으로 압축
