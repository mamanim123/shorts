# 카메라 샷 강제 적용 완료 (Two-shot, Three-shot 우선)

**작성일**: 2025-12-31  
**작업명**: 카메라 샷 강제 적용 (단독샷 → 두샷/쓰리샷 전환)  
**상태**: ✅ 완료

---

## 🎯 작업 목표

### 문제 상황
- 현재 이미지 프롬프트가 **단독샷(Single shot) 위주**로 생성됨
- 두샷(Two-shot), 쓰리샷(Three-shot) 등 **다양한 카메라 구도가 부족**
- 캐릭터 간 상호작용을 보여주는 샷이 거의 없음

### 해결 방안
- **캐릭터 수에 따라 자동으로 적절한 카메라 샷 선택**
- 2명 이상 등장 시 **Two-shot, Three-shot 우선 적용**
- 씬마다 다양한 카메라 앵글 순환 (Over-the-shoulder, Dutch angle 등)

## ✅ 추가된 함수

### `ensureCameraShot` (라인 987-1039)

```typescript
const ensureCameraShot = (prompt: string, sceneNumber: number, characterCount: number): string => {
  // 이미 카메라 샷이 있으면 건너뛰기
  const hasCameraShot = /\b(two-shot|three-shot|over-the-shoulder|...)\b/i.test(prompt);
  if (hasCameraShot) return prompt;

  // 캐릭터 수에 따른 카메라 샷 선택
  let cameraShots: string[] = [];

  if (characterCount >= 3) {
    // 3명 이상: Three-shot 우선
    cameraShots = [
      "Three-shot framing all characters",
      "Wide three-shot with dynamic composition",
      "Over-the-shoulder three-shot",
      ...
    ];
  } else if (characterCount === 2) {
    // 2명: Two-shot 우선
    cameraShots = [
      "Two-shot with eye contact",
      "Over-the-shoulder two-shot",
      "Medium two-shot capturing both characters",
      ...
    ];
  } else {
    // 1명: 다양한 단독 샷
    cameraShots = [
      "Medium close-up shot",
      "Over-the-shoulder perspective",
      "Dutch angle close-up",
      ...
    ];
  }

  // 씬 번호에 따라 순환하며 선택
  const selectedShot = cameraShots[sceneNumber % cameraShots.length];
  return `${selectedShot}, ${prompt}`;
};
```

## 🎬 카메라 샷 종류

### Three-shot (3명 이상)
1. **Three-shot framing all characters** - 모든 캐릭터를 프레임에 담음
2. **Wide three-shot with dynamic composition** - 넓은 구도로 역동적인 구성
3. **Over-the-shoulder three-shot** - 어깨 너머 3명 샷
4. **Medium three-shot capturing interaction** - 중간 거리에서 상호작용 포착
5. **Close-up three-shot focusing on expressions** - 표정에 집중하는 클로즈업
6. **Dutch angle three-shot for tension** - 긴장감을 위한 더치 앵글

### Two-shot (2명)
1. **Two-shot with eye contact** - 눈 맞춤이 있는 두샷
2. **Over-the-shoulder two-shot** - 어깨 너머 두샷
3. **Medium two-shot capturing both characters** - 두 캐릭터를 모두 담는 중간 샷
4. **Close-up two-shot with physical interaction** - 신체 접촉이 있는 클로즈업
5. **Wide two-shot showing full bodies** - 전신을 보여주는 넓은 두샷
6. **Dutch angle two-shot for dramatic tension** - 드라마틱한 긴장감을 위한 더치 앵글

### Single shot (1명)
1. **Medium close-up shot** - 중간 클로즈업
2. **Over-the-shoulder perspective** - 어깨 너머 시점
3. **Dutch angle close-up** - 더치 앵글 클로즈업
4. **Extreme close-up on face** - 얼굴 극단 클로즈업
5. **Wide shot with environment** - 환경을 포함한 넓은 샷
6. **Low angle shot for emphasis** - 강조를 위한 로우 앵글
