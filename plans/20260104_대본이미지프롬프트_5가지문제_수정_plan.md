### 5. 장르 폴백 추가 (Medium)

**파일**: `services/genreGuidelines.ts`

**현재 문제**:
```ts
const getGenreGuideline = (genreId: string) => {
  const genre = genres.find(g => g.id === genreId);
  return genre || null;  // ⚠️ null 반환 → 장르 완전 손실
};
```

**수정 후**:
```ts
const getGenreGuideline = (genreId: string): GenreGuideline | null => {
  // 'none' 또는 빈 문자열은 null 반환 (의도적 제외)
  if (!genreId || genreId === 'none') {
    return null;
  }

  // 장르 찾기
  const genre = genres.find(g => g.id === genreId);

  // 못 찾으면 기본값 폴백 (affair-suspicion)
  if (!genre) {
    console.warn(`[genreGuidelines] 장르 ID "${genreId}"를 찾을 수 없습니다. 기본값(affair-suspicion) 사용`);
    return genres.find(g => g.id === 'affair-suspicion') || null;
  }

  return genre;
};
```

**예상 효과**:
- ✅ 커스텀 장르 손실 시에도 안정적 동작
- ✅ 기본 장르로 폴백
- ✅ 콘솔 경고로 디버깅 용이
