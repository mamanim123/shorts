## 🎯 2. "선택안함" 선택 시 상세 동작

### 2-1. 전체 흐름

```
"0. 선택안함" 선택
  ↓
resolveEffectiveOutfits() 호출
  ↓
resolveOutfit("0. 선택안함") → "" (빈 문자열 반환)
  ↓
pickRandomOutfit() 호출
  ↓
SHORTFORM_OUTFIT_LIST (100+ 의상)에서 완전 랜덤 선택
  ↓
주인공 의상: 랜덤 선택된 의상 (예: "White Halter-neck Knit + Red Micro Mini Skirt")
조연 의상: 주인공과 다른 것 중 랜덤 (예: "Navy Off-shoulder Tight Knit + Blue Denim Mini Skirt")
  ↓
buildFinalPrompt()에 전달
  ↓
AI에게 "이 의상을 모든 씬에서 동일하게 사용하세요" 명시
```

### 2-2. resolveEffectiveOutfits() 함수 (라인 1851-1880)

```tsx
const resolveEffectiveOutfits = () => {
  // Step 1: 사용자 선택값
  const override = femaleOutfit && femaleOutfit.trim().length > 0
    ? femaleOutfit
    : "0.";

  // Step 2: 기본값 카테고리 (메인화면 설정)
  const fallbackCategory = defaultSettings?.category || "0.";

  // Step 3: "0."로 시작하면 fallback 사용, 아니면 사용자 선택 유지
  const effectiveInput = (override.startsWith("0.") && fallbackCategory)
    ? fallbackCategory
    : override;

  // Step 4: 의상 해석
  let resolvedFemale = resolveOutfit(effectiveInput)?.trim() ||
                       pickRandomOutfit() ||
                       DEFAULT_FEMALE_OUTFIT;

  // Step 5: 주인공 의상 확정
  const primaryFemaleOutfit = resolvedFemale;

  // Step 6: 조연 의상 (주인공과 다른 것)
  const secondaryFemaleOutfit = pickRandomOutfit([primaryFemaleOutfit]) ||
                                DEFAULT_FEMALE_OUTFIT2;

  return {
    lockedFemaleOutfit: primaryFemaleOutfit,
    lockedFemaleOutfit2: secondaryFemaleOutfit,
    lockedMaleOutfit: resolvedMale
  };
};
```

### 2-3. resolveOutfit() 함수 (라인 2160-2171)

```tsx
function resolveOutfit(outfitInput: string) {
  if (!outfitInput || outfitInput.trim() === '') return "";

  // "0."으로 시작 → AI DECIDE 신호
  if (outfitInput.startsWith("0.")) return "";

  // "1-4."로 시작 → pickV3Item() 호출 (카테고리 조합)
  if (outfitInput.startsWith("1.") || outfitInput.startsWith("2.") || ...) {
    return pickV3Item(outfitInput);
  }

  // 그 외: 수동 입력값 그대로 반환
  return outfitInput;
}
```

**"0. 선택안함" 처리**:
- `resolveOutfit("0. 선택안함")` → **빈 문자열 `""`** 반환
- 이는 "AI가 자동으로 선택하라"는 신호

### 2-4. pickRandomOutfit() 함수 (라인 1855-1861)

```tsx
const pickRandomOutfit = (exclude: string[] = []) => {
  const usable = SHORTFORM_OUTFIT_LIST
    .map(o => o.name)
    .filter(name => name && !exclude.includes(name));

  if (usable.length === 0) {
    return 'White Halter-neck Knit + Red Micro Mini Skirt';
  }
  return pickRandom(usable);  // 완전 랜덤 선택
};
```

**동작**:
1. SHORTFORM_OUTFIT_LIST (100+ 의상)에서 의상 이름만 추출
2. `exclude` 배열에 없는 의상만 필터링
3. 필터링된 리스트에서 **완전 랜덤** 선택
4. 리스트가 비었으면 기본값 반환

## 🔄 3. "선택안함" vs 카테고리 선택 비교

| 구분 | 0. 선택안함 | 1-4. 카테고리 선택 |
|------|-------------|-------------------|
| **의상 선택 풀** | SHORTFORM_OUTFIT_LIST (100+ 개) | 특정 카테고리 (MODERN_CHIC 등) |
| **선택 방식** | 사전 정의된 완성 의상 | Top + Bottom 조합 또는 Dress |
| **추가 요소** | 없음 | Material + Detail 추가 |
| **매번 결과** | 랜덤 (재선택 시 바뀜) | 랜덤 (매번 새로 생성) |
| **의상 예시** | "White Halter-neck Knit + Red Micro Mini Skirt" | "High-gloss Satin White Halter-neck Knit + Red Micro Mini Skirt with Gold zipper detail" |
| **제어성** | 낮음 (완전 랜덤) | 중간 (카테고리 지정) |

### 카테고리 선택 시 동작 (예: "1. 모던 시크")

**pickV3Item() 함수** (라인 2123-2158):
```tsx
function pickV3Item(category: string): string {
  // 카테고리 결정
  let collection = OUTFIT_COLLECTIONS.MODERN_CHIC;
  if (category.startsWith('2')) collection = OUTFIT_COLLECTIONS.GLAMOUR_PARTY;
  if (category.startsWith('3')) collection = OUTFIT_COLLECTIONS.ACTIVITY_LUXURY;

  // 40% 드레스, 60% 투피스
  const useDress = Math.random() < 0.4;
  let itemBase = "";

  if (useDress && collection.dresses.length > 0) {
    itemBase = pickRandom(collection.dresses);
  } else {
    const top = pickRandom(collection.tops);
    const bottom = pickRandom(collection.bottoms);
    itemBase = `${top} + ${bottom}`;
  }

  // 소재 및 디테일 추가
  const material = pickRandom(V3_MATERIALS);
  const detail = pickRandom(V3_DETAILS);

  return `${material} ${itemBase} with ${detail}`;
}
```

**결과 예시**:
- "High-gloss Satin White Sleeveless Turtleneck + Navy Leather Micro Mini Skirt with Gold zipper detail"
- "Matte Leather Navy Blue Textured Tweed Deep V-neck Fitted Mini Dress with Pearl buttons"
- "Metallic Knit White See-through Blouse + Royal Blue Fitted Mini Skirt with Asymmetric cut"

매번 **재료, 의상, 디테일이 랜덤 조합**되어 완전히 새로운 의상 생성

## ✅ 6. 핵심 결론

### 6-1. "선택안함" 선택 시 실제 동작

**질문**: "선택안함"을 선택하면 어떻게 되나요?

3. ✅ **선택된 의상은 모든 씬(6-8개)에서 동일하게 사용** (고정)
4. ✅ **AI에게 명시적으로 전달** (AI가 임의로 변경 불가)

### 6-2. 의상 고정 메커니즘

**중요**: 의상은 **FrontEnd에서 한 번만 결정**됩니다!

```tsx
// generateScript() 실행 시 1회만 호출
const { lockedFemaleOutfit, lockedFemaleOutfit2, lockedMaleOutfit } =
  resolveEffectiveOutfits();
```

- 같은 대본 생성 요청 내에서는 **의상이 절대 변하지 않음**
- 재실행 시에는 `resolveEffectiveOutfits()`가 다시 호출되어 **새로운 랜덤 의상** 선택 가능

### 6-3. 폴백 계층 구조

```
1순위: 사용자 UI 선택 (femaleOutfit)
       ↓ (없거나 "0."로 시작)
2순위: defaultSettings.category (메인화면 설정)
       ↓ (없거나 "0."로 시작)
3순위: pickRandomOutfit() (완전 랜덤)
       ↓ (SHORTFORM_OUTFIT_LIST가 비었을 경우)
4순위: DEFAULT_FEMALE_OUTFIT (하드코딩 기본값)
       = "White Halter-neck Knit + Red Mini Skirt"
```
