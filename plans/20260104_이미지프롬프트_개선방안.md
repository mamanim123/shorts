## 🚨 2. 현재 문제점

### 2.1 LLM이 생성한 실제 응답 (문제 사례)

**Scene 7 (Two-shot 누락)**:
```
Scene 7. Slot A — stunning 40대 Korean woman, Slot B — friend, ...
```
→ ❌ "Two-shot" 누락 (2명인데 구도 없음)

**Scene 8 (핏 태그 일부 누락)**:
```
Scene 8. ... tight-fitting premium tailored design, ...
```
→ ❌ "bodycon silhouette", "form-fitting elegant attire" 2개 누락

### 2.2 왜 이런 일이 발생하는가?

#### 원인 1: 프롬프트 토큰 길이 제약
- 장르 가이드 (약 500 토큰) + scriptImage 템플릿 (약 2000 토큰) = **총 2500+ 토큰**
- LLM이 응답 생성 시 **토큰 제약**으로 일부 지침 생략

#### 원인 2: 지침 우선순위 불명확
```
[장르 가이드라인] (대본 구조 강조)
     +
[scriptImage 템플릿] (이미지 프롬프트 강조)
```
→ LLM이 **장르 가이드에 집중**하느라 세부 지침(Two-shot, tight-fitting) 놓침

#### 원인 3: 검증 후 재시도 메커니즘 부족
- 현재: 1회 재시도만 허용 (MAX_VALIDATION_RETRIES=1)
- 재시도 시 **명확한 오류 피드백 부족**

#### 원인 4: 후처리 단계에서 강제 삽입 없음
- 검증만 하고 **자동으로 수정하지 않음**
- LLM 재생성만 의존

## 💡 3. 개선 방안 (우선순위별)

### 🔴 우선순위 1: 필수 지침을 프롬프트 최상단에 배치

#### 현재 구조:
```
[장르 가이드라인]
═══════════════
[scriptImage 템플릿]
  - CRITICAL WARNING (여기에 Two-shot, tight-fitting)
  - 필수 구성 요소
  - 예시들
```

#### 개선 구조:
```
[🚨 ABSOLUTE MANDATORY RULES - NEVER OMIT 🚨]
1. 2명 이상 = "Two-shot" 또는 "Three-shot" 필수 (맨 앞!)
2. 핏 태그 3종 모두 필수:
   - tight-fitting premium tailored design
   - bodycon silhouette accentuating feminine curves
   - form-fitting elegant attire with body-conscious refinement
3. no text, no letters, --ar 9:16 필수

⚠️ These rules override ALL other instructions! ⚠️
═══════════════
[장르 가이드라인]
═══════════════
[scriptImage 템플릿]
```

**효과**:
- LLM이 **가장 먼저** 필수 규칙을 읽음
- "override ALL other instructions" → 최우선 적용

### 🟡 우선순위 5: 후처리 단계에서 필수 태그 강제 삽입

#### 개선 방안:

**파일**: `youtube-shorts-script-generator.tsx` (postProcessScripts 함수 수정)

**새로운 함수 추가**:
```typescript
const forceInsertMandatoryTags = (longPrompt: string, scene: Scene) => {
  let updated = longPrompt;

  // 1. Two-shot/Three-shot 강제 삽입
  if (scene.characterIds && scene.characterIds.length >= 2) {
    const angle = scene.characterIds.length === 2 ? 'Two-shot' : 'Three-shot';
    if (!updated.toLowerCase().includes('two-shot') && !updated.toLowerCase().includes('three-shot')) {
      // Scene N. 다음에 강제 삽입
      updated = updated.replace(/^(Scene \d+\.\s*)/, `$1${angle}, `);
    }
  }

  // 2. 핏 태그 강제 삽입 (1개라도 있으면 스킵)
  const hasFitTag = REQUIRED_FIT_TAGS.some(tag => updated.toLowerCase().includes(tag.toLowerCase()));
  if (!hasFitTag) {
    // 첫 번째 핏 태그 추가
    updated += ' Tight-fitting premium tailored design.';
  }

  // 3. no text, --ar 9:16 강제 삽입
  if (!updated.includes('no text')) {
    updated += ' no text, no letters';
  }
  if (!updated.includes('--ar 9:16')) {
    updated += ' --ar 9:16';
  }

  return updated;
};

// postProcessScripts에서 호출
scenes.forEach(scene => {
  if (scene.longPrompt) {
    scene.longPrompt = forceInsertMandatoryTags(scene.longPrompt, scene);
  }
});
```

**효과**:
- LLM이 놓친 필수 태그를 **자동으로 추가**
- 검증 실패 가능성 **0%**로 감소

```typescript
const forceInsertMandatoryTags = (scene: Scene): Scene => {
  if (!scene.longPrompt) return scene;

  let updated = scene.longPrompt;

  // 1. Two-shot/Three-shot 강제 삽입
  if (scene.characterIds && scene.characterIds.length >= 2) {
    const angle = scene.characterIds.length === 2 ? 'Two-shot' : 'Three-shot';
    const hasAngle = /two-shot|three-shot/i.test(updated);

    if (!hasAngle) {
      // Scene N. 다음에 강제 삽입
      updated = updated.replace(/^(Scene \d+\.\s*)/, `$1${angle}, `);
      console.warn(`[forceInsert] Scene ${scene.sceneNumber}: ${angle} 자동 삽입`);
    }
  }

  // 2. 핏 태그 강제 삽입 (1개라도 있으면 스킵)
  const REQUIRED_FIT_TAGS = [
    'tight-fitting premium tailored design',
    'bodycon silhouette accentuating feminine curves',
    'form-fitting elegant attire with body-conscious refinement'
  ];

  const hasFitTag = REQUIRED_FIT_TAGS.some(tag => updated.toLowerCase().includes(tag.toLowerCase()));
  if (!hasFitTag) {
    // 실사화 태그 앞에 삽입
    const fitTag = 'Tight-fitting premium tailored design, ';
    if (updated.includes('Raw photo style')) {
      updated = updated.replace('Raw photo style', fitTag + 'Raw photo style');
    } else {
      updated += ' ' + fitTag.trim() + '.';
    }
    console.warn(`[forceInsert] Scene ${scene.sceneNumber}: 핏 태그 자동 삽입`);
  }

  // 3. no text 강제 삽입
  if (!updated.toLowerCase().includes('no text')) {
    updated += ', no text, no letters';
    console.warn(`[forceInsert] Scene ${scene.sceneNumber}: no text 자동 삽입`);
  }

  // 4. --ar 9:16 강제 삽입
  if (!updated.includes('--ar 9:16')) {
    updated += ' --ar 9:16';
    console.warn(`[forceInsert] Scene ${scene.sceneNumber}: --ar 9:16 자동 삽입`);
  }

  return { ...scene, longPrompt: updated };
};
```

**postProcessScripts 함수에서 호출**:

const postProcessScripts = (scripts: any[], options: PostProcessOptions) => {
  return scripts.map(script => {
    const scenes = Array.isArray(script.scenes) ? script.scenes : [];

    // 각 씬에 필수 태그 강제 삽입
    const processedScenes = scenes.map(scene => forceInsertMandatoryTags(scene));

    return {
      ...script,
      scenes: processedScenes
    };
  });
};
