# 20260104_이미지프롬프트_실행계획_plan

이미지 프롬프트의 지침 준수율을 100%로 끌어올리기 위한 핵심 로직(검증 완화, 재시도 강화, 자동 보정)을 실제 코드에 반영하는 계획입니다.

## 📋 주요 수정 사항

### 1. [services/promptValidator.ts] 검증 로직 완화
- **내용**: 핏 태그 3종 중 1개만 포함되어도 검증을 통과하도록 수정합니다.
- **이유**: LLM의 부담을 줄이고 생성 성공률을 높이기 위함입니다.

### 2. [youtube-shorts-script-generator.tsx] 재시도 메커니즘 강화
- **내용**: 
    - `MAX_VALIDATION_RETRIES`를 1회에서 **3회**로 상향합니다.
    - 재시도 시 LLM에 전달하는 오류 피드백 문구를 더 강력하고 명확하게 개선합니다.

### 3. [youtube-shorts-script-generator.tsx] 자동 보정(forceInsertMandatoryTags) 도입
- **내용**: 
    - LLM이 생성한 결과물에 필수 태그(`Two-shot`, `핏 태그`, `no text`, `--ar 9:16`)가 누락된 경우, 코드가 이를 자동으로 감지하여 적절한 위치에 강제 삽입합니다.
- **이유**: LLM의 가끔 발생하는 실수를 시스템적으로 완벽하게 보완하여 검증 통과율 100%를 보장합니다.
