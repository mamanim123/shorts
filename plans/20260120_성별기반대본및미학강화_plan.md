# 성별 기반 대본 생성 및 여성 캐릭터 미학 강화 계획 (2026-01-20)

## 목표 (Goal)
- 사용자가 선택한 성별(여성/남성)에 따라 대본의 주인공이 결정되도록 로직을 개편합니다.
- 모든 여성 캐릭터가 마마님의 취향에 맞는 화려하고 아름다운 외모(Glamorous & Beautiful)로 생성되도록 프롬프트 미학을 강화합니다.
- 남성 캐릭터 선택 시 이미지 프롬프트에 성별이 올바르게 반영되지 않는 버그를 수정합니다.

## 체크리스트 (Checklist)
- [ ] `labPromptBuilder.ts`: `LabScriptOptions`에 `gender` 필드 추가
- [ ] `labPromptBuilder.ts`: `buildLabScriptPrompt`에서 성별 기반 주인공 지시 로직 구현
- [ ] `labPromptBuilder.ts`: 여성 캐릭터 미학 키워드 강화 (`PROMPT_CONSTANTS.FEMALE_BODY` 등)
- [ ] `ShortsLabPanel.tsx`: `handleAiGenerate`에서 선택된 성별을 프롬프트 빌더에 전달
- [ ] `ShortsLabPanel.tsx`: `enforceKoreanIdentity`에서 하드코딩된 `woman` 제거 및 성별 변수 적용
- [ ] 수동 검증: 여성 주인공 대본 생성 확인
- [ ] 수동 검증: 남성 주인공 대본 생성 확인
- [ ] 수동 검증: 이미지 프롬프트 내 성별 및 미학 키워드 적용 확인

## 문제 정의
현재 쇼츠랩의 "한국인 정체성" 설정은 이미지 프롬프트 보강용으로만 사용되며, 대본 생성 시 주인공의 성별을 반영하지 못하고 있습니다. 또한 여성 캐릭터의 미학적 수준을 마마님의 취향에 맞게 더욱 화려하고 아름답게 고정할 필요가 있습니다.

## 제안된 변경 사항

### 1. 대본 생성 로직 (주인공 성별 반영)

#### [MODIFY] [labPromptBuilder.ts](file:///f:/test/쇼츠대본생성기-v3.5.1/services/labPromptBuilder.ts)
- `LabScriptOptions` 인터페이스에 `gender: 'female' | 'male'` 필드를 추가합니다.
- `buildLabScriptPrompt` 함수에서 이 성별 정보를 사용하여 AI에게 주인공 성별을 명시적으로 지시합니다.
    - 예: `📌 주인공 성별: ${gender === 'female' ? '여성' : '남성'}`
    - AI에게 해당 성별의 시점에서 이야기를 전개하도록 시스템 프롬프트를 수정합니다.

#### [MODIFY] [ShortsLabPanel.tsx](file:///f:/test/쇼츠대본생성기-v3.5.1/components/ShortsLabPanel.tsx)
- `handleAiGenerate` 함수에서 `buildLabScriptPrompt`를 호출할 때 `settings.koreanGender` 값을 전달하도록 수정합니다.

### 2. 여성 캐릭터 미학 강화 (화려하고 아름다운 외모 고정)

#### [MODIFY] [labPromptBuilder.ts](file:///f:/test/쇼츠대본생성기-v3.5.1/services/labPromptBuilder.ts)

### 3. 이미지/영상 프롬프트 성별 버그 수정

## 검증 계획

### 수동 검증
1. **여성 주인공 테스트**: "한국인" 활성 + "여성" 선택 후 대본 생성 시 여성이 주인공인 대본이 나오는지 확인.
2. **남성 주인공 테스트**: "한국인" 활성 + "남성" 선택 후 대본 생성 시 남성이 주인공인 대본이 나오는지 확인.
3. **미학 확인**: 생성된 프롬프트에 강화된 미학 키워드(glamorous, stunning 등)가 포함되어 있는지 확인.
4. **남성 프롬프트 확인**: 남성 선택 시 이미지 프롬프트에 `man` 키워드가 정상적으로 들어가는지 확인.
