# 목표(Goal): 겨울철 의상 보정 로직 정교화 (배꼽 노출 유지 및 팔만 긴팔화)

마마님께서 지시하신 대로, 겨울/눈 주제일 때 상의를 긴팔로 바꾸더라도 **배꼽티(Crop Top)나 튜브탑(Tube Top) 등 배꼽이 보이는 노출 특성은 그대로 유지**하면서 팔만 타이트한 긴팔로 보정하도록 로직을 정교화합니다.

## 체크리스트(Checklist)
- [ ] 노출형 상의(Crop, Tube Top 등) 키워드 식별 로직 설계 [ ]
- [ ] `adjustOutfitForSeason` 함수 업데이트: 노출 부위는 남기고 팔만 긴팔로 변환 [ ]
- [ ] 변환된 결과가 "긴팔 배꼽티(Long-sleeved Crop Top)"와 같은 명확한 형상이 되도록 보장 [ ]
- [ ] 테스트 스크립트를 통한 다양한 노출 의상 변환 결과 검증 [ ]

---

## 1. 개요
기존 로직은 상의 노출 키워드를 단순히 `Tight-fitting Long-sleeve`로 대체하여 배꼽 노출 정보가 사라질 가능성이 있었습니다. 이를 수정하여 **"노출은 하되, 팔만 따뜻하게(긴팔)"**라는 마마님의 미학을 코드로 구현합니다.

## 2. 정교화된 보정 로직

### 2.1 대상 상의 키워드
- `Crop Top`, `Tube Top`, `Crop Tank`, `Bustier`, `Corset`, `Camisole`, `Sports Bra` 등 배꼽이나 몸매가 드러나는 아이템들.

### 2.2 변환 방식
1.  **키워드 유지**: 의상 명칭에 `Crop`, `Tube`, `Bustier` 등의 단어가 있으면 이를 삭제하지 않고 유지합니다.
2.  **팔만 추가**: 
    - `Halter-neck` -> `Long-sleeved Halter-neck` (또는 `High-neck Long-sleeve`)
    - `Sleeveless` -> `Long-sleeved`
    - `Short-sleeve` -> `Long-sleeve`
    - `Tube Top` -> `Long-sleeved Tube Top` (팔만 토시처럼 붙은 형태 묘사)
3.  **결과 예시**:
    - "White Halter-neck Tight Crop Top" -> "White **Long-sleeved** Halter-neck Tight **Crop Top**"
    - "White Off-shoulder Tube Top" -> "White **Long-sleeved** Off-shoulder **Tube Top**"
    - "Pink Satin Bustier Top" -> "Pink Satin **Long-sleeved Bustier Top**"

## 3. 구현 계획
- `labPromptBuilder.ts` 내의 `adjustOutfitForSeason` 함수를 정규표현식 및 조건문을 활용하여 '의상의 본체 형태'는 보존하고 '소매 길이'만 수정하도록 재작성합니다.
- 긴팔화된 의상 앞에도 항상 `Tight-fitting`을 붙여 마마님께서 원하시는 슬림한 라인을 유지합니다.
