# 목표(Goal): 여성 캐릭터 계절별 의상 자동 보정 로직 구현 (겨울/눈 대비)

마마님께서 포착하신 '눈 오는 날 반팔 입는 문제'를 해결하기 위해, 주제(Topic)에 '겨울'이나 '눈'이 포함될 경우 상의를 자동으로 **타이트한 긴팔**로 변환하는 스마트 로직을 구현합니다.

## 체크리스트(Checklist)
- [ ] 주제(Topic)에서 겨울/눈 키워드 감지 로직 설계 [ ]
- [ ] 의상 텍스트 변형 함수(`adjustOutfitForSeason`) 구현 [ ]
- [ ] `labPromptBuilder.ts`의 여성 의상 선택 과정에 계절 보정 적용 [ ]
- [ ] 보정된 의상이 `lockedOutfits`, `characters`, `scenes` 전체에 일관되게 적용되는지 확인 [ ]
- [ ] 최종 결과물에서 의상의 형태(긴팔 등)가 명시적으로 프롬프트에 포함되도록 개선 [ ]

---

## 1. 개요
현재 의상 리스트(`UNIFIED_OUTFIT_LIST`)는 고정된 문자열로 되어 있어, 눈이 오는 배경임에도 '민소매(Sleeveless)'나 '반팔(Short-sleeve)' 의상이 선택될 수 있습니다. 이를 해결하기 위해 주제를 분석하여 계절에 맞지 않는 상의 노출을 자동으로 긴팔로 보정합니다.

## 2. 주요 로직 (스마트 보정)

### 2.1 계절 감지
- 주제(`topic`)에 "눈", "겨울", "눈발", "Snow", "Winter" 등의 키워드가 포함되어 있는지 확인합니다.

### 2.2 의상 변환 로직 (`adjustOutfitForSeason`)
겨울일 경우, 여성 의상 명칭에서 다음 키워드를 찾아 변환합니다:
- `Sleeveless` -> `Tight-fitting Long-sleeve`
- `Short-sleeve` -> `Tight-fitting Long-sleeve`
- `Halter-neck` -> `High-neck Tight Long-sleeve`
- `Tube Top` -> `Tight-fitting Long-sleeve Layered`
- `Off-shoulder` -> `Tight-fitting Long-sleeve`
- 만약 긴팔(Long-sleeve)이라는 명시가 없다면, 상의 아이템(Knit, Blouse, Polo, Shirt 등) 앞에 `Tight-fitting Long-sleeve`를 강제로 삽입합니다.

### 2.3 볼륨감(Bust) 유지
상의가 긴팔로 바뀌더라도 마마님께서 강조하신 **"가슴이 크고 아름다운"** 실루엣이 프롬프트에서 누락되지 않도록 `Tight-fitting`, `Bodycon`, `Accentuating body lines` 태그를 더욱 강력하게 결합합니다.

## 3. 적용 범위
- `ManA`, `ManB`는 이미 골프 주제 시 골프복을 입도록 수정되었으나, 남성에게도 겨울용 긴팔 보정을 동일하게 적용하여 일관성을 높이겠습니다.
- 여성 캐릭터(`WomanA`, `WomanB`)의 모든 이미지 프롬프트(`longPrompt`, `shortPrompt`)에 보정된 의상 명칭을 100% 동일하게 사용합니다.
