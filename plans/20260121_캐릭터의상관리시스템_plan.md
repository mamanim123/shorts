# 목표(Goal): 캐릭터 및 의상 관리 시스템 구현

마마님의 지시에 따라 쇼츠랩 우측 패널에 캐릭터 선택/관리 및 의상 추출 기능을 추가합니다.

## 체크리스트(Checklist)

### 기획 (Planning)
- [ ] UI/UX 목업 확정
- [ ] 데이터 구조 설계 (캐릭터 JSON 스키마)

### UI 구현 (Execution)
- [ ] 3개 메인 탭 구현 (캐릭터 선택 / 캐릭터 관리 / 의상 추출)
- [ ] 캐릭터 선택 탭: 저장된 캐릭터 목록 및 슬롯 할당 UI
- [ ] 캐릭터 관리 탭: 얼굴/헤어/체형 서브탭 및 프리셋 관리
- [ ] 의상 추출 탭: 이미지 업로드 및 Vision AI 연동
- [ ] 캐릭터 저장/불러오기 기능

### 백엔드 (Backend)
- [ ] 캐릭터 데이터 저장 API (characters.json)
- [ ] Vision API 연동 (의상 프롬프트 추출)
- [ ] 대본 생성 시 선택된 캐릭터 프롬프트 적용

### 검증 (Verification)
- [ ] 캐릭터 저장/불러오기 테스트
- [ ] 의상 추출 정확도 테스트
- [ ] 대본 생성 시 캐릭터 적용 테스트

---

## 1. 탭 구조

| 탭 | 기능 |
|----|------|
| **캐릭터 선택** | 저장된 캐릭터 목록에서 선택 → 슬롯(Woman A, Man A 등)에 할당 |
| **캐릭터 관리** | 새 캐릭터 생성/수정 (얼굴, 헤어, 체형 서브탭) |
| **의상 추출** | 이미지 업로드 → AI가 의상 프롬프트 추출 → 의상 리스트에 추가 |

## 2. 캐릭터 관리 서브탭

| 서브탭 | 내용 |
|--------|------|
| **얼굴** | 이미지 업로드 → AI가 얼굴 특징 추출 (쌍꺼풀, 턱선, 피부톤 등) |
| **헤어** | 프리셋 선택 또는 직접 입력 (long soft-wave, short bob 등) |
| **체형** | 프리셋 선택 (키 큰 슬림, 아담하고 글래머러스, 건강미 탄탄 등) |

## 3. 데이터 구조 (characters.json)

```json
{
  "characters": [
    {
      "id": "char-001",
      "name": "김지영",
      "age": "40대",
      "gender": "female",
      "face": "A stunning Korean woman, double eyelid, sharp jawline, fair skin",
      "hair": "long soft-wave hairstyle",
      "body": "slim hourglass figure with toned body, curvy feminine figure, glamorous silhouette",
      "createdAt": "2026-01-21T22:00:00Z"
    }
  ]
}
```

## 4. 워크플로우

```
[캐릭터 관리 탭]        [캐릭터 선택 탭]           [대본 생성]
얼굴/헤어/체형 설정 → 저장 → 캐릭터 선택 → 슬롯 할당 → 프롬프트에 자동 반영
```
