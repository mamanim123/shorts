## 1. 탭 구조

| 탭 | 기능 |
|----|------|
| **캐릭터선택** | 저장된 캐릭터 목록에서 선택 → 슬롯(Woman A, Man A 등)에 할당 |
| **캐릭터관리** | 새 캐릭터 생성/수정 (얼굴, 헤어, 체형 서브탭) |
| **의상추출** | 이미지 업로드 → AI가 의상 프롬프트 추출 → 의상 리스트에 추가 |

---

## 2. 캐릭터관리 서브탭

| 서브탭 | 내용 |
|--------|------|
| **얼굴** | 이미지 업로드 → AI가 얼굴 특징 추출 (쌍꺼풀, 턱선, 피부톤 등) |
| **헤어** | 프리셋 선택 또는 직접 입력 (long soft-wave, short bob 등) |
| **체형** | 프리셋 선택 (키 큰 슬림, 아담하고 글래머러스, 건강미 탄탄 등) |

---

## 3. 의상 리스트 기능

- **인라인 리스트**: 오른쪽 패널에 항상 표시
- **추가**: 드롭다운에서 프리셋 선택 또는 직접 입력
- **삭제**: 각 항목 옆 🗑️ 버튼
- **의상 추출**: 이미지 업로드 시 AI가 분석하여 자동 추가
- **저장**: `outfits.json`에 영구 저장

## 5. 워크플로우

```
[캐릭터관리]           [캐릭터선택]           [의상추출]           [대본생성]
얼굴/헤어/체형 설정 → 캐릭터 저장 → 슬롯 할당 → 의상 선택 → 프롬프트에 자동 반영
```
