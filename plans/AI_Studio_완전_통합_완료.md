## 🐛 알려진 이슈

### 1. Web Speech API 제한
- 브라우저마다 지원 음성 다름
- 긴 텍스트 끊김 가능
- 오프라인 미작동

**해결 방법**: Google Cloud TTS API 사용 (추후 구현)

### 2. FFmpeg 설치 필요
- 영상 합성 기능 사용 시 필수
- Windows PATH 설정 필요

**해결 방법**: 설치 가이드 제공

### 3. 대용량 처리
- 많은 프레임 (10개 이상) 시 시간 소요
- 메모리 사용량 증가

**해결 방법**: 프레임 수 제한 (최대 12개)
