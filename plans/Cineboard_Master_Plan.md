# 🎬 Cineboard (김시네) 완전 구현 마스터 플랜

## 🎯 목표
`김씨네프롬프트.txt`에 명시된 "상상을 시각적으로 가이드하는 연출 도구"로서의 씨네보드 기능을 완성한다. 단순히 대본을 요약하는 수준을 넘어, 인물의 일관성을 유지하고 전문적인 영상미를 구현할 수 있는 시스템을 구축한다.

---

## 🛠️ Phase 1: AI 인물 분석 (Character Vibe)
*   **핵심**: 이름만 찾는 것이 아니라, 대본을 분석하여 인물의 특징을 AI가 자동으로 제안한다.
*   **세부 작업**:
    *   `/api/analyze-characters` 엔드포인트 신설 또는 기존 `/api/generate` 확장.
    *   대본 전체를 AI에게 전달하여 {이름, 나이대, 성별, 주요 의상 스타일, 성격적 특징}을 JSON으로 추출.
    *   UI: 추출된 정보를 `characterNotes`에 자동으로 채워주는 인터페이스.

## 🎥 Phase 2: 전문 연출 프롬프트 고도화
*   **핵심**: "조선 시대 고증 질감", "스튜디오 조명" 등의 설명이 실제 카메라/조명 키워드로 변환된다.
*   **세부 작업**:
    *   `STYLE_PRESETS`에 기술적 프롬프트(Technical Keywords) 매핑 (Lens: 35mm, 85mm / Lighting: Volumetric, Rim light 등).
    *   프롬프트 빌더에서 스타일 선택 시 해당 키워드를 최우선으로 배치.

## 📽️ Phase 3: 장면별 비디오 생성 지원
*   **핵심**: 정지 영상을 넘어 동영상 가이드라인을 제공한다.
*   **세부 작업**:
    *   `CineboardScene` 타입에 `videoUrl`, `isVideoGenerating` 필드 추가.
    *   장면 카드에 '영상 만들기' 버튼 추가.
    *   `/api/video/ai-generate` (또는 기존 연동) 호출 인터페이스 구축.

## 🔒 Phase 4: Identity Lock 및 프롬프트 최적화
*   **핵심**: "오른쪽 눈 밑 점" 같은 특징이 영문 프롬프트로 자동 변환되어 모든 장면에 고정된다.
*   **세부 작업**:
    *   사용자가 입력한 `Note`(한글)를 이미지 생성 직전 AI를 통해 영문 묘사(Visual Descriptor)로 변환하는 유틸리티 추가.
    *   `Identity Lock` 강화: 캐릭터별 이미지를 참조(Casting)할 때 해당 이미지의 텍스트 묘사도 함께 유지.
