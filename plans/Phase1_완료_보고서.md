## 🐛 알려진 이슈

### 1. Web Speech API 제한사항
- 브라우저마다 지원하는 음성이 다름
- 긴 텍스트는 끊길 수 있음
- 오프라인에서는 작동하지 않음

**해결 방법**: Google Cloud TTS API 사용 (추후 구현)

### 2. FFmpeg 설치 필요
- 영상 합성 기능 사용 시 FFmpeg 필수
- Windows에서 PATH 설정 필요

**해결 방법**: 설치 가이드 제공

### 3. 대용량 영상 처리
- 많은 프레임 (10개 이상) 처리 시 시간 소요
- 메모리 사용량 증가

**해결 방법**: 프레임 수 제한 (최대 12개)
