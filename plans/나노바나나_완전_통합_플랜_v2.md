#### 1-3. 신 만들기 (영상 합성) (4일)
**목표**: 생성된 프레임들을 영상으로 합성

**구현 위치**: `ai_studio_bundle/components/SceneCreator.tsx` (신규)

**기능**:
- 프레임 → 영상 전환 (각 프레임 3초)
- 카메라 효과 (Zoom, Pan 등)
- 트랜지션 (Fade, Slide)
- 여러 영상 합성
- 배경 음악 추가

**서버 API** (`server/index.js`):
```javascript
const ffmpeg = require('fluent-ffmpeg');

app.post('/api/create-scene', upload.single('bgMusic'), async (req, res) => {
  const { frames, duration, transition, cameraEffect } = JSON.parse(req.body.config);
  const bgMusic = req.file;
  
  const tempDir = path.join(__dirname, 'temp', Date.now().toString());
  fs.mkdirSync(tempDir, { recursive: true });
  
  try {
    // 프레임 저장
    frames.forEach((frame, i) => {
      const buffer = Buffer.from(frame.replace(/^data:image\/\w+;base64,/, ''), 'base64');
      fs.writeFileSync(path.join(tempDir, `frame_${i}.png`), buffer);
    });
    
    const outputPath = path.join(tempDir, 'scene.mp4');
    
    let command = ffmpeg();
    
    // 각 프레임을 duration만큼 표시
    frames.forEach((_, i) => {
      command = command
        .input(path.join(tempDir, `frame_${i}.png`))
        .inputOptions([`-t ${duration}`]);
    });
    
    // 카메라 효과 적용
    if (cameraEffect === 'zoom') {
      command = command.videoFilters('zoompan=z=\'min(zoom+0.0015,1.5)\':d=125');
    } else if (cameraEffect === 'pan') {
      command = command.videoFilters('crop=iw/2:ih/2:(in-out_time)*iw/10:0');
    }
    
    // 트랜지션 적용
    if (transition === 'fade') {
      const filterComplex = [];
      for (let i = 0; i < frames.length - 1; i++) {
        filterComplex.push(
          `[${i}:v][${i+1}:v]xfade=transition=fade:duration=0.5:offset=${duration * i + duration - 0.5}[v${i}]`
        );
      }
      command = command.complexFilter(filterComplex);
    }
    
    // 배경 음악
    if (bgMusic) {
      const musicPath = path.join(tempDir, 'music.mp3');
      fs.writeFileSync(musicPath, bgMusic.buffer);
      command = command.input(musicPath).audioFilters('volume=0.5');
    }
    
    command
      .outputOptions(['-c:v libx264', '-pix_fmt yuv420p', '-preset fast', '-crf 23'])
      .output(outputPath)
      .on('end', () => {
        res.sendFile(outputPath, () => {
          setTimeout(() => fs.rmSync(tempDir, { recursive: true, force: true }), 5000);
        });
      })
      .on('error', (err) => {
        console.error('FFmpeg error:', err);
        res.status(500).json({ error: 'Scene creation failed' });
        fs.rmSync(tempDir, { recursive: true, force: true });
      })
      .run();
  } catch (error) {
    console.error('Scene creation error:', error);
    res.status(500).json({ error: 'Scene creation failed' });
    fs.rmSync(tempDir, { recursive: true, force: true });
  }
});
```

```javascript
const textToSpeech = require('@google-cloud/text-to-speech');

app.post('/api/text-to-speech', async (req, res) => {
  const { text, voice, emotion } = req.body;
  
  const client = new textToSpeech.TextToSpeechClient();
  
  const request = {
    input: { text },
    voice: {
      languageCode: 'ko-KR',
      name: voice || 'ko-KR-Standard-A',
      ssmlGender: 'NEUTRAL'
    },
    audioConfig: {
      audioEncoding: 'MP3',
      pitch: emotion === 'happy' ? 2 : emotion === 'sad' ? -2 : 0,
      speakingRate: emotion === 'angry' ? 1.2 : 1.0
    }
  };
  
  try {
    const [response] = await client.synthesizeSpeech(request);
    res.set('Content-Type', 'audio/mpeg');
    res.send(response.audioContent);
  } catch (error) {
    console.error('TTS error:', error);
    res.status(500).json({ error: 'TTS failed' });
  }
});
```

## 🎬 완전 자동화 워크플로우

### 시나리오: 쇼츠 영상 자동 제작

```
1. 쇼츠 생성기에서 대본 생성
   ↓
2. 캐릭터 이미지 1장 업로드 (참조용)
   ↓
3. "프레임 자동 생성" 버튼 클릭
   ↓
4. AI가 6컷 이미지 자동 생성 (캐릭터 일관성 유지)
   ↓
5. 필요시 이미지 편집 (얼굴 보정, 색상 변경 등)
   ↓
6. "영상 만들기" 버튼 클릭
   ↓
7. 카메라 효과 및 트랜지션 자동 적용
   ↓
8. 대본 텍스트 → 음성 변환 (TTS)
   ↓
9. 영상 + 음성 합성
   ↓
10. 완성된 쇼츠 영상 다운로드!
```

**예상 소요 시간**: 5-10분 (수동 작업 시 2-3시간)

### 1. 원클릭 쇼츠 제작 버튼
```typescript
// youtube-shorts-script-generator.tsx
const handleOneClickShortsCreation = async (story: StoryResponse) => {
  // 1. 프레임 생성
  const frames = await generateFrames(story);
  
  // 2. 영상 합성
  const video = await createScene(frames);
  
  // 3. 음성 생성
  const audio = await textToSpeech(story.scriptBody);
  
  // 4. 최종 합성
  const finalVideo = await mergeVideoAndAudio(video, audio);
  
  // 5. 다운로드
  downloadVideo(finalVideo);
};
```
