## 🎯 핵심 발견사항

### ✅ 100% 공통점 (모든 대본에 존재)

#### 1. **이중의미 (Double Entendre) 구조**
**발견**: 18개 대본 **전부** 이중의미 사용!

**패턴**:
- 야한 것처럼 들리지만 → 건전한 반전
- 청자의 오해 유도 → 마지막에 반전

**예시**:
```
"어머, 끝이 왜 이렇게 굵어요?"
→ 넥타이 얘기였음

"너무 예뻐서 오히려 남자 경험이 없는..."
→ 첫날밤 배 쓰다듬기 (낮게 = 목소리)

"다리가 셋이라 그런가 봐요"
→ 화장실 빨리 가는 이유
```

#### 2. **대사 비율: 60~80%**
**발견**: 나레이션보다 **대사가 압도적으로 많음**!

**현재 시스템**: 나레이션 90% + 대사 10%
**대박 대본**: 나레이션 20~40% + 대사 60~80%

**구조**:
```
짧은 상황 설명 (나레이션 2~3문장)
↓
대사 중심 전개 (5~10개 대사)
↓
반전 대사 (1~2개)
```

#### 3. **Hook: 질문형 + 충격형**
**발견**: 첫 문장이 **즉시 호기심 유발**

**패턴**:
- 질문형: "코가 좀 많이 크시네요. 그럼 엄지 발가락도 크신가요?"
- 충격형: "제가요 제가 임신이라뇨?"
- 상황형: "너무나 예뻐서 오히려 남자 경험이 없는..."

**현재 시스템**: Hook 전략 없음
**대박 대본**: 첫 문장부터 강력한 Hook

#### 4. **3단 반전 구조**
**발견**: 모든 대본이 **최소 2~3번 반전**

**구조**:
```
1단: 초기 오해 유발 (야한 상황처럼 보임)
2단: 중간 반전 (아, 이게 아니구나)
3단: 최종 반전 (완전히 다른 의미였음)
```

**예시 (편도선 수술)**:
```
1단: "정상적인 부부 관계는 언제부터 가능할까요?"
→ (청자: 어? 야한 얘기인가?)

2단: "그쪽을 많이 써야 하는 일이라서요"
→ (청자: 확실히 야한 얘기네!)

3단: "편도선 수술 받은 환자한테는..."
→ (청자: 아! 목구멍 얘기였구나! ㅋㅋㅋ)
```

#### 5. **감정 자극: 웃음 + 놀라움**
**발견**: 모든 대본이 **강한 감정 2가지 이상** 유발

**감정 조합**:
- 웃음 + 놀라움 (90%)
- 웃음 + 당황 (80%)
- 웃음 + 공감 (70%)

#### 6. **길이: 40~60초 (12~18문장)**
**발견**: 대부분 **50초 내외**

**현재 시스템**: 50초 내외 (15~20문장) ✅ 일치!

## 📈 대박 대본의 공통 패턴

### 패턴 1: "오해 → 확신 → 반전" 구조

**18개 중 18개 사용 (100%)**

```typescript
[구조]
1. 오해 유발 문장 (야한 것처럼)
2. 오해 강화 문장 (확실히 야한 거 맞네)
3. 오해 강화 문장 (더 야하게)

```

```
2. "다른 건 그 자리에서 바로 따 먹어야 제맛"

4. "과일 얘기였음!"
```


## 🎬 실전 예시 비교

### 대박 대본 스타일 (과수원)

```
[나레이션] 휴가 나온 아들이 과수원 아주머니를 도왔다.

[대사 - 아들] "아 과수원 아주머니께서 사과 말고도 
                따먹을 거 많으니 모든 셀 거 따먹으라 하셨어요"
[대사 - 아버지] "그런데 왜 사과만 가져왔어?"
[대사 - 아들] "다른 건 그 자리에서 바로 따 먹어야 제맛이래서요"

[나레이션] 며칠 뒤 아들이 전화했다.

[대사 - 아들] "아버지, 휴가 때 그 과수원에서 
                바로 따먹던 맛이 자꾸 생각나서 힘듭니다"

[나레이션] 아버지는 과수원을 찾아갔다.

[대사 - 아주머니] "안타깝지만 그건 우리 총각이 
                   하도 씹컷 따먹어서 지금은 시간 말랐답니다"
[대사 - 아주머니] "그때쯤이면 아주 먹음직스럽게 
                   익어서 딱 벌어져 있을 테니까요"

[대사 - 아버지] "그런데 어찌 아들 휴가 때에 맞춰서 
                 과일이 다 익을 거라고 아십니까?"
[대사 - 아주머니] "글쎄요 아버님 그건 따는 사람 나름 아니겠어요? 
                   우리 총각이 워낙 잘 따더라고요"
```

**분석**:
- 대사 비율: 약 75%
- 반전 횟수: 3회
- Hook: "따먹을 거 많으니" (즉시 오해 유발)
- 감정: 웃음 + 놀라움

### 현재 시스템 스타일 (예상)

