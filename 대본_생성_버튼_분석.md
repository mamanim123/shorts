- **파일 위치**: `f:\test\쇼츠대본생성기-v6.1.0\youtube-shorts-script-generator.tsx`
- **버튼 라벨**: `대본 {scriptCount}개 생성하기` (약 3707번 라인)

- **주요 기능**:

    - 이미지 프롬프트(Scene 1~12) 동시 생성.
    - 장면 규칙 자동 검증 로직 포함.

## 2. 메인 앱 하단 액션 버튼 (App.tsx)
왼쪽 설정 패널 하단에 위치한 버튼들입니다.

- **파일 위치**: `f:\test\쇼츠대본생성기-v6.1.0\App.tsx`
- **버튼 목록**:
    1. **"JSON 불러오기"** (1209번 라인)
        - **연결 함수**: `handleGenerate` -> `importStoryFromJson`
        - **용도**: 외부 JSON 데이터를 대본으로 변환.
    2. **"대본→이미지 생성"** (1218번 라인)
        - **연결 함수**: `handleGenerate`
        - **용도**: 기존 대본 텍스트를 기반으로 이미지 프롬프트만 생성.
    3. **"스토리 생성"** (1228번 라인)
        - **연결 함수**: `handleGenerate` -> `generateStory`
        - **용도**: 이미지 없이 텍스트 스토리만 생성 (구형 로직).
    4. **"쇼츠 대본 생성"** (1237번 라인)
        - **동작**: `shorts-generator` 탭으로 이동 및 자동 생성 플래그 활성화.
        - **용도**: 메인 쇼츠 생성기를 실행하는 진입점.
    5. **"쇼츠 기본 (고정 룰)"** (1250번 라인)
        - **동작**: 단순화된 템플릿(`__useSimpleTemplate`)을 사용하여 쇼츠 생성기 실행.
    6. **"쇼츠 실험 (다양성 강화)"** (1265번 라인)
        - **동작**: 테스트용 템플릿(`__useTestTemplate`)을 사용하여 쇼츠 생성기 실행.
