제가 생각하는 쇼츠 대본 생성 방식

  1. 훅은 "설명문" 절대 금지

  현재 문제:
  ❌ "함박눈이 펑펑 내리는 날인데도 이 언니들 고집은 아무도 못 말려."
  ❌ "언니들 오늘 필드에서 진짜 큰일 날 뻔했잖아요!"
  → 설명하고 있음. 지루함.

  제가 만든다면:
  ✅ "야, 너 지금 뭐 하는 거야?" (대화로 바로 시작)
  ✅ 혜진이 주머니에 손을 넣었다. (행동으로 시작)
  ✅ 이게 진짜 남편 비상금이었어? (질문으로 시작)
  ✅ 죽는 줄 알았어. (충격 한 방)

  2. 구조는 5가지 패턴을 랜덤 믹스

  현재는 1가지 구조만 강제하고 있습니다. 저라면 최소 5가지 구조를 준비하고 AI가 선택하게 합니다:

  패턴 A: 역순 구조 (반전부터 시작)

  1. 반전 결과 공개 (충격)
  2. "왜 이렇게 됐냐면..." (과거 회상)
  3. 빌드업
  4. 반전 재확인
  예시: "알고 보니 군고구마였어. 왜냐면..."

  패턴 B: 이중 반전

  1. 훅
  2. 첫 번째 반전
  3. "근데 여기서 끝이 아니야"
  4. 두 번째 진짜 반전
  예시: "비상금인 줄 알았는데 편지였어. 그런데 그 편지 내용이..."

  패턴 C: 대화 중심

  1. 대화로 시작
  2. 대화 속 힌트
  3. 대화 중 반전
  4. 대화로 마무리
  예시:
  "언니, 그거 뭐야?"
  "이거? 그냥..."
  "설마 남편 거 아니지?"
  "...맞아."

  패턴 D: 관찰자 시점

  1. 3인칭 관찰 (카메라 앵글)
  2. 수상한 행동 포착
  3. 줌인
  4. 정체 공개

  패턴 E: 질문 연쇄

  1. 질문 1
  2. 질문 2
  3. 질문 3
  4. 대답 (반전)

  ---
  3. 반전은 7가지 카테고리로 분산

  현재: 물건 오해 5번 반복

  제가 한다면:

  | 카테고리   | 예시                   |
  |--------|----------------------|
  | 물건 반전  | 명품 → 골프공 (현재)        |
  | 관계 반전  | 바람 → 생일 서프라이즈        |
  | 정체성 반전 | 남편 친구 → 알고 보니 오빠     |
  | 상황 반전  | 위기 → 기회 (골프 내기에서 이김) |
  | 역할 반전  | 당하는 사람 → 실은 주도한 사람   |
  | 목적 반전  | 로맨스 → 실은 사업          |
  | 감정 반전  | 슬픔 → 기쁨, 분노 → 웃음     |

  AI에게 지시:
  "매 대본마다 위 7가지 반전 카테고리 중 하나를 랜덤 선택하세요.
  절대 3번 이상 같은 카테고리 사용 금지."

  ---
  4. 장소/배경은 20가지 풀에서 랜덤

  현재: 골프장 100%

  제가 한다면:
  장소 풀 = [
    "골프장", "카페", "백화점", "호텔 로비", "피팅룸",
    "주차장", "엘리베이터", "루프탑", "와인바", "요가 스튜디오",
    "미용실", "네일샵", "갤러리", "북카페", "스파",
    "골프 연습장", "테니스장", "수영장", "레스토랑", "펜트하우스"
  ]

  AI에게: "{{TOPIC}}과 어울리는 장소를 위 리스트에서 1개 선택하세요."

  ---
  5. 종결어미 다양화 (10가지 이상)

  현재: ~했어, ~했지, ~했더라 (90%)

  제가 한다면:
  [감정별 종결어미 풀]

  놀람: ~하네, ~하잖아, ~했다니까
  확신: ~했어, ~한 거야, ~했다고
  추측: ~했을까, ~한 것 같아, ~했겠지
  회상: ~했더라, ~했었어, ~했었지
  강조: ~했거든, ~했단 말이야, ~했다니까
  의문: ~할까, ~한 거야?, ~했어?
  부정: ~안 했어, ~아니야, ~한 게 아니야
  청유: ~하자, ~해봐, ~할래?

  AI에게: "다양한 종결어미를 섞어 쓰세요. 같은 어미 3번 연속 금지."

  ---
  6. 톤은 5가지 중 선택

  현재: 항상 40대 여성 구어체로 고정

  제가 한다면:

  | 톤    | 특징       | 예시                    |
  |------|----------|-----------------------|
  | 속삭임  | 비밀 공유 느낌 | "내가 너한테만 말해줄게"        |
  | 흥분   | 에너지 넘침   | "진짜 미쳤어! 완전 대박!"      |
  | 차분   | 관찰자 시점   | "그때 혜진은 알았다."         |
  | 코믹   | 과장, 리액션  | "아니 이게 뭔 소리야ㅋㅋ"       |
  | 서스펜스 | 긴장감 유지   | "그리고... 그 손에 들린 건..." |

  ---
  7. 문장 수는 "유연하게"

  현재: 12문장 강제 (하지만 실제론 17-23문장)

  제가 한다면:
  - 짧은 대본: 6-8문장 (펀치라인 중심)
  - 중간 대본: 10-12문장 (밸런스)
  - 긴 대본: 15-18문장 (스토리 중심)

  AI가 장르와 주제에 따라 선택하도록.

  🎯 실제 프롬프트 예시

  제가 만든다면 이렇게 지시하겠습니다:

  **[쇼츠 대본 생성 규칙]**

  1. **구조**: 아래 5가지 중 1개 랜덤 선택
     - 역순 구조
     - 이중 반전
     - 대화 중심
     - 관찰자 시점
     - 질문 연쇄

  2. **훅 스타일**: 아래 중 1개 선택 (설명문 절대 금지!)
     - 대화로 시작
     - 행동으로 시작
     - 질문으로 시작
     - 충격 한 문장
     - 일상 묘사 → 돌발

  3. **반전 카테고리**: 7가지 중 랜덤 (이전 대본과 다르게)
     - 물건, 관계, 정체성, 상황, 역할, 목적, 감정

  4. **장소**: 20가지 풀에서 {{TOPIC}}에 맞게 선택

  5. **종결어미**: 10가지 이상 섞어서 사용 (3번 연속 금지)

  6. **톤**: 속삭임/흥분/차분/코믹/서스펜스 중 1개

  7. **문장 수**: 6-18문장 (자유롭게)

  **중요**: 이전 대본과 80% 이상 유사하면 자동 재생성!
