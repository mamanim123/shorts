## 🎯 작동 방식

### 대본 생성 시점
1. 사용자가 "AI 대본 생성" 클릭
2. `buildLabScriptPrompt`가 호출되어 AI에 프롬프트 전송
3. AI가 JSON 응답 생성 (scenes 배열 포함)
4. 각 씬에 `videoPrompt` 필드가 포함되어야 함
5. `handleAiGenerate`가 JSON 파싱
6. 파싱된 scenes에 videoPrompt 포함되어 상태 업데이트
7. 미리보기 탭으로 자동 전환

### 비디오 프롬프트 생성
1. 사용자가 "스마트 비디오 프롬프트 생성" 버튼 클릭
2. `handleRefineVideoPrompt` 호출
3. 서버 `/api/video/refine-prompt`로 요청 전송:
   - script: 전체 대본
   - scriptLine: 씬 텍스트
   - action: 요약
   - emotion: 빈 문자열
   - visualPrompt: 이미지 프롬프트
   - targetAge: 선택한 나이 (예: "40대")
   - characterSlot: 샷 타입
4. 서버에서 AI가 분석하고 videoPrompt 생성
5. 생성된 videoPrompt가 씬에 저장되고 VIDEO 탭에 표시
