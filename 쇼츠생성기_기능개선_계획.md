# 쇼츠 생성기 대본 목록 기능 개선

## 문제점

1. **대본 클릭 시 표시 안됨**: 쇼츠 생성기 오버레이 내부에는 메인 화면의 오른쪽 패널이 없어서 대본을 클릭해도 아무것도 표시되지 않음
2. **삭제/즐겨찾기 기능 없음**: 저장된 대본을 관리할 수 없음

## 해결 방법

### 옵션 1: 대본 클릭 시 메인 화면으로 이동 (권장)

**장점:**
- 메인 화면의 OutputDisplay 컴포넌트를 활용하여 대본을 완전하게 표시
- 기존 UI/UX와 일관성 유지
- 구현이 간단

**작동 방식:**
1. 왼쪽 패널에서 대본 클릭
2. 쇼츠 생성기 오버레이 닫기
3. 메인 화면의 보관함 탭으로 이동
4. 해당 대본을 오른쪽 패널에 표시

### 옵션 2: 쇼츠 생성기 오른쪽에 대본 뷰어 추가

**단점:**
- 복잡한 구현 필요
- 쇼츠 생성기와 대본 뷰어가 동시에 표시되어 혼란스러움

## 작동 방식

1. **대본 클릭**
   - 쇼츠 생성기 오버레이 닫기
   - 메인 화면의 보관함 탭으로 이동
   - 해당 대본을 오른쪽 패널에 표시

2. **즐겨찾기**
   - 별 아이콘 클릭
   - 즐겨찾기 상태 토글
   - 메인 화면의 보관함에도 반영

3. **삭제**
   - 휴지통 아이콘 클릭
   - 확인 메시지 표시
   - 삭제 후 목록에서 제거
