# 쇼츠 생성기 저장된 대본 표시 기능 구현

## 목표

쇼츠 생성기 왼쪽 패널에서 저장된 대본을 클릭하면 오른쪽 패널에 해당 대본과 분석 결과를 표시합니다.

## 구현 방법

### 0. types.ts 수정 - 분석 결과 타입 추가

**StoryResponse에 analysis 필드 추가:**
```typescript
export interface StoryResponse {
  // 기존 필드들...
  analysis?: {
    scores: {
      hook: number;
      twist: number;
      emotion: number;
      length: number;
      viral: number;
    };
    improvements: string[];
    strengths: string[];
    totalScore: number;
  };
}
```

### 1. ShortsScriptGenerator 컴포넌트 수정

**Props 추가:**
```typescript
interface ShortsScriptGeneratorProps {
  onSave?: (scriptData: any) => void;
  selectedStory?: StoryResponse | null;  // 새로 추가
}
```

**표시 로직:**
- `selectedStory`가 있으면 → 저장된 대본 표시
- `selectedStory`가 없으면 → 생성된 대본 3개 표시

## 작동 방식

1. **대본 생성**
   - 쇼츠 생성기에서 대본 3개 생성
   - 오른쪽 패널에 표시

2. **대본 저장**
   - "보관함에 저장" 버튼 클릭
   - 왼쪽 패널 목록에 추가

3. **저장된 대본 보기**
   - 왼쪽 패널에서 대본 클릭
   - 오른쪽 패널에 해당 대본 표시
   - "대본 생성으로 돌아가기" 버튼으로 복귀

4. **대본 관리**
   - 즐겨찾기: 별 아이콘 클릭
   - 삭제: 휴지통 아이콘 클릭
