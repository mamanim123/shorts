## 작동 방식

1. **쇼츠 생성기 열기**
   - 메인 화면에서 "🧪 쇼츠 생성기 (Dev)" 버튼 클릭
   - 전체 화면 오버레이로 표시

2. **대본 생성 및 저장**
   - 오른쪽 패널에서 대본 생성
   - "보관함에 저장" 버튼 클릭
   - 왼쪽 패널에 즉시 표시됨
   - 메인 화면의 보관함에도 자동 저장

3. **저장된 대본 확인**
   - 왼쪽 패널에서 대본 클릭
   - 쇼츠 생성기를 닫고 메인 화면으로 이동
   - 오른쪽 패널에 대본 내용 표시

4. **연속 작업**
   - 쇼츠 생성기 화면은 계속 유지
   - 저장하지 않은 대본도 오른쪽 패널에서 계속 볼 수 있음
   - 분석 결과도 유지됨

## 개선 사항

### 이전
- 쇼츠 생성기 왼쪽 패널에 닫기 버튼만 있음
- 저장된 대본을 확인하려면 메인 화면으로 돌아가야 함
- 저장하지 않은 대본이 사라질 위험

### 개선 후
- ✅ 왼쪽 패널에 저장된 대본 목록 실시간 표시
- ✅ 쇼츠 생성기 화면에서 바로 확인 가능
- ✅ 저장하지 않은 대본과 분석 결과 유지
- ✅ 메인 화면의 보관함에도 자동 저장
