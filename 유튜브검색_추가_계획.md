# 유튜브 쇼츠 검색 기능 추가

## 목표

설정 화면에서 불필요한 옵션을 숨기고, "유튜브 쇼츠 검색" 버튼을 추가하여 오른쪽 패널에 쇼츠영상검색.html을 표시합니다.

## 요구사항

1. **ConfigPanel.tsx 수정**
   - 재생성참고문구 (useRegenerationGuidance) 숨기기
   - 글래머러스 핏 (safeGlamour) 숨기기
   - 대박모드 (useViralMode) 숨기기
   - "유튜브 쇼츠 검색" 버튼 추가

2. **App.tsx 수정**
   - 오른쪽 패널에 새로운 탭 추가 ('youtube-search')
   - 버튼 클릭 시 해당 탭으로 전환
   - iframe으로 쇼츠영상검색.html 표시

## 구현 방법

### 1. ConfigPanel.tsx 수정

**숨길 옵션들:**
```typescript
// 주석 처리 또는 조건부 렌더링으로 숨기기
// - useRegenerationGuidance
// - safeGlamour  
// - useViralMode
```

**추가할 버튼:**
```typescript
<button
  onClick={onOpenYoutubeSearch}
  className="w-full py-3 bg-gradient-to-r from-red-600 to-pink-600 text-white rounded-xl font-bold hover:shadow-xl transition-all"
>
  🔍 유튜브 쇼츠 검색
</button>
```

### 2. App.tsx 수정

**오른쪽 탭에 'youtube-search' 추가:**
```typescript
const [rightTab, setRightTab] = useState<'analysis' | 'shortform' | 'longform' | 'youtube-search'>('analysis');
```

**탭 버튼 추가:**
```typescript
<button 
  className={`px-4 py-2 text-sm ${rightTab === 'youtube-search' ? 'bg-slate-800 text-purple-300' : 'text-slate-400 hover:text-slate-200'}`} 
  onClick={() => setRightTab('youtube-search')}
>
  유튜브 검색
</button>
```

**iframe으로 HTML 표시:**
```typescript
{rightTab === 'youtube-search' && (
  <div className="h-full">
    <iframe
      src="쇼츠영상검색.html"
      className="w-full h-full border-0"
      title="유튜브 쇼츠 검색"
    />
  </div>
)}
```
