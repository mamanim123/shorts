## 🎯 주요 기능

### 1. 캐릭터별 의상 규칙 관리
- **여성 캐릭터** (4명):
  - Female A (지영): 롱웨이브, 세련된 룩
  - Female B (혜경): 단발, 매력적인 스타일
  - Female C: 하이포니테일, 운동적 우아함
  - Female D (캐디): 하이번, 밝고 전문적
- **남성 캐릭터** (3명):
  - Male A (준호): 단정한 단발, 댄디함
  - Male B (민수): 깔끔한 숏컷, 친근함
  - Male C: 사이드파트, 비즈니스 우아함

### 2. 각 슬롯 관리 필드
- `identity`: "A stunning Korean woman" / "A handsome Korean man"
- `hair`: 헤어스타일 설명
- `body`: 체형 및 신체 특징 설명
- `ageLabel`: "in her 40s" / "in his 40s"
- `style`: 스타일 및 분위기
- `outfitFit`: 의상 핏 설명

### 3. 공통 설정
- `negativePrompt`: 부정 프롬프트
- `qualityTags`: 품질 태그
- `defaultFemaleAge`: 기본 여성 나이
- `defaultMaleAge`: 기본 남성 나이

### 4. 백업 시스템
- 최대 5개 백업 저장
- 백업 생성/복구/삭제/이름변경
- 백업 내용 직접 편집 가능
- localStorage 영구 저장

## 🔧 사용 방법

### UI에서 의상 규칙 수정
1. 쇼츠랩 패널 열기
2. 장르 관리 모달 열기 (톱니바퀴 아이콘)
3. "의상규칙" 탭 클릭
4. 원하는 캐릭터 슬롯 선택 (Female A~D, Male A~C, Common)
5. 필드 수정 (identity, hair, body 등)
6. "저장" 버튼 클릭

### 코드에서 의상 규칙 사용
```typescript
import { getCharacterRules } from './services/shortsLabCharacterRulesManager';

const rules = getCharacterRules();
const femaleAIdentity = rules.femaleA.identity; // "A stunning Korean woman"
const femaleAHair = rules.femaleA.hair;         // "long soft-wave hairstyle"
const femaleABody = rules.femaleA.body;         // "perfectly managed..."
```

### 백업 생성 및 복구
1. 의상규칙 탭에서 "백업" 버튼 클릭
2. 백업 이름 입력 (선택)
3. 백업 목록에서 복구할 백업 선택
4. "복원" 버튼 클릭

### 기본값으로 초기화
1. "기본값으로 초기화" 버튼 클릭
2. 확인 메시지에서 "확인" 클릭
3. 모든 설정이 DEFAULT_CHARACTER_RULES로 복구됨
