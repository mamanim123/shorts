### ✅ **적용되고 있는 규칙**

#### 1. 나이 언급 강제 (`ensureAgeMention`)
**위치**: `services/geminiService.ts` 라인 816-830

**기능**:
- AI가 생성한 프롬프트에 나이 정보가 없으면 자동으로 추가
- `Korean woman in her 40s` 형태로 강제 삽입

**적용 시점**:
- 라인 1121: `scene.longPrompt`에 적용
- 라인 1124: `scene.shortPrompt`에 적용

**코드**:
```typescript
const ensureAgeMention = (text: string, targetAge?: string): string => {
  if (!targetAge) return text;
  const agePattern = /in (her|his|their) (20s|30s|40s|50s|60s)/i;
  if (agePattern.test(text)) return text;
  
  // 나이 정보가 없으면 "Korean woman" 뒤에 추가
  return text.replace(
    /(Korean woman|Korean man)/i,
    `$1 in her ${targetAge}`
  );
};
```

**해결 방법**: `engine_config.json`의 프롬프트에 명시되어 있지만, AI가 이를 따르지 않을 수 있음

#### 2. 슬롯별 헤어 스타일 고정 ❌
**문제**: 씬마다 헤어 스타일이 변경되는지 검증하는 로직 없음

**영향**: Slot A가 1번 씬에서는 긴 웨이브, 2번 씬에서는 단발로 바뀔 수 있음

**해결 방법**: 후처리 검증 로직 필요

#### 3. 의상 일관성 검증 ❌
**문제**: 1번 씬부터 8번 씬까지 의상이 동일한지 검증하는 로직 없음

**영향**: 씬마다 의상이 바뀔 수 있음

**해결 방법**: 후처리 검증 로직 필요

#### 4. 의상 핏 키워드 강제 ❌
**문제**: `tight-fitting, bodycon, accentuating body lines` 자동 추가 로직 없음

**영향**: AI가 이 키워드를 누락할 수 있음

**해결 방법**: 후처리에서 자동 추가 필요

#### 5. 타 국적 금지 검증 ❌
**문제**: `Western`, `Japanese`, `Chinese` 등이 포함되었는지 검증하는 로직 없음

**영향**: AI가 실수로 타 국적을 언급할 수 있음

**해결 방법**: 후처리 검증 및 제거 로직 필요
