## ✅ 충돌 여부 분석

### **결론: 충돌 없음! ✅**

#### 1. **레이어 분리가 명확함**
- **프로그램 로직**: `WomanA` (ID)
- **UI 표시**: `Woman A` (Label) 또는 `Female A` 모두 가능
- **의상 규칙**: `femaleA` (Key)

#### 2. **매핑 로직이 이미 존재**
- `geminiService.ts:1276-1277`: WomanA → femaleA 매핑 명시
- `ShortsLabPanel.tsx:100-113`: 양쪽 표기 모두 지원

#### 3. **상호 변환 가능**
```typescript
// ID → 의상 규칙 키
function idToRuleKey(id: string): string {
  return id.replace('Woman', 'female').replace('Man', 'male')
             .replace(/^(female|male)/, (m) => m.toLowerCase());
}

// 예시:
idToRuleKey('WomanA') // → 'femaleA'
idToRuleKey('ManB')   // → 'maleB'
```
